**第194章 哪来的信球货？**

「闪电琴行」

“昨晚来的时候没注意到..这个店名还真是摇滚啊。”，站在店前，林穗安忍不住咽了口唾沫。

“因为闪烁的父亲叫闪电。”，苏屿川解释道。

“哦？”，林穗安白了他一眼，“苏苏酱调查的还真是清楚呢，hentai～”

...

与Guitar House的纤尘不染，琳琅满目不同，透过「闪电琴行」外玻璃门就能清楚的看到店内一片狼藉，泡面桶，饮料瓶丢的满地都是，打眼一扫甚至连一块干净的地方都找不到。

“像是被打劫过了。”，苏屿川眼睛瞪的大大的，一度怀疑自己来错了地方。

“进去真的很需要勇气呢..”，林穗安有些哆嗦。

“先等等吧，我问问闪烁是不是这家，万一搬迁了呢。”，苏屿川示意林穗安稍安勿躁，说罢便打开了手机。

自从上次为了要琴行地址加了闪烁的联系方式后，两人一句交流都没有过。

虽然贸然联系有些不礼貌，但苏屿川也别无他法了。

【予川】：你的地址是对的吗？

【予川】：我好像走到了传销窝点。

【twinkle】：没问题啊？你拍张照片我看看。

【予川】：行。

二人聊天期间，林穗安的视线始终没离开过苏屿川的手机屏幕。

可就在苏屿川调出摄像头准备对着门匾拍一张时，琴行大门却被从里面推开了。

“哪来的信球货？拍你舅姥姥啊拍？”

说话的是一位不修边幅的大汉，来势汹汹，嗓门大得很，虎背熊腰络腮胡，手臂上还有纹身，一看就不是好惹的。

“咿呀！”，林穗安吓了一大跳，下意识躲到了苏屿川的身后。

“对不起对不起我错了我错了，老大老大！别嘎我腰子！”，苏屿川手机都吓掉了，差点跪下来磕头认错，“无意冒犯..是闪烁让我来的！”

听到闪烁的名字后，大叔明显顿了顿，狠戾的气场也收了起来。

“妮儿的朋友？进来吧。”

就这样，苏屿川和林穗安悻悻的跟着闪电走进了店内。

不看不知道一看吓一跳，越往里走越是不能直视，最后甚至演变成了不能并排走的惨状。

顶里面就是修理工作台了，只有两张课桌大小，比起Guitar House的应有尽有一应俱全，闪电的工作台只能用寒酸来形容。

“吉他给俺瞅瞅。”，坐下之后，闪电指了指林穗安背上的吉他包。

“好..好的！”，林穗安颤颤巍巍的取下了琴包。

在闪电观察吉他的时候，苏屿川也观察起了他。

太权威了！长相太权威了！

非要形容的话，那就是【完美符合人们对于非主流摇滚乐手的刻板印象】

他的工作台周边满是烟蒂和啤酒瓶，空气质量堪比垃圾焚烧厂，老鼠和蟑螂肯定是没有的，因为这个鸟不拉屎的地方根本孕育不了生命。

“我的吉他还能修吗？”，眼见闪电面露为难，林穗安紧张的询问道。

“...”，闪电沉默了片刻，“估计是梅兰芳睡觉了。”

“啥意思？”，林穗安不解其意。

“没戏。”，苏屿川叹了一口气。

“狗屁！”，闪电回怼了苏屿川一句，“是「老子登场」！”

“歇后语不是这么乱用的吧大叔？”，苏屿川被逗乐了。

但也挺好，闪电敢这么说应该是有修好的把握的，如此想来，这位大叔倒没有那么吓人了。

“你们是跟俺闺女一起玩摇滚的吗？”，闪电放下了吉他，转过头朝苏屿川发问。

“不是，她是我在江宁音乐节上认识的朋友，我们的吉他摔坏了，她说找您能修好。”，苏屿川不卑不亢道。

“从江宁来的？那还挺远的。”，闪电有些诧异，“你说俺知道了。”

“啊？”，苏屿川挠了挠脑袋，“俺知道了。”

“俺说俺知道了！”，闪电被这个外地的小伙子惹得心烦。

“哦哦好！”，苏屿川连连应答，“闪叔叔也是玩摇滚乐的吗？”

“儿戏的玩意儿。”，闪电顿了顿，旋即嗤笑道，“屁用没有。”

“可是您左手上还有摇滚纹身呢。”

苏屿川注意到了闪电左臂胳膊上的纹身是“摇滚不死”，只是不知道为什么，“不”字显得很淡。

“洗过一次，太疼，俺喊停了。”，闪电点燃一根香烟，使得乌烟瘴气房间爆发了生化危机，“后来就只洗了「不」字，洗了好几次，还是怕疼。”

“秧秧，你出去帮我买瓶水。”，见闪电开始抽烟，苏屿川轻轻扯了扯林穗安的衣角打发她出去。

他可不想让秧秧吸二手烟。

“哦..好。”

林穗安小心翼翼的出门了。

“叔叔，这琴修好大概要多少钱？”，苏屿川心里没底道，“应该..花不到七千吧？”

虽然闪烁说了会给友情价，但毕竟损坏的太严重，想要修好人力物力都少不了，他已经做了最坏的打算。

“你个信球货把俺这当黑店啊？！”，闪电勃然大怒，“俺闺女说了，不收你们钱。”

“这不好吧？”

苏屿川属实没想到闪烁说的「友情价」是分文不取。

中原妮子太性情了！

“给俺一天时间，俺能给你修好。”，闪电没搭理苏屿川，自顾自说道。

“真的吗叔？”，苏屿川倍感惊喜。

闪电的语气可是把握十足啊！

“俺发现你这个信球货话特别多。”，闪电烦躁的摆了摆手，“俺修琴的技术全国独一档，你把心放肚子里。”

“可是叔....你的工具呢？”，苏屿川在工作台上环视一圈，愣是连个螺丝刀都没看到。

“在桌底下。”，闪电淡淡应道，说罢还用脚往底下踢了一脚。

苏屿川的视线随他跟了上去。

电锯，榔头，锤子..

这对吗？！

这真的是修吉他用的吗？这难道不是干木匠用的吗？

“叔..真的没问题吗？”，苏屿川有些后悔病急乱投医的这个决定了，万一林穗安的吉他最后尸骨无存，他怕是要内疚一辈子。

“有问题俺再去洗一次纹身！”，闪电掏出锯子，一边大刀阔斧的拆解吉他一边问道，“话说你们下午有安排没有？”

“暂且没有。”，苏屿川看着光头强同款的电锯被搬上桌，整个人都开始颤抖。

“俺给你个地址，你帮俺把这东西送给一个人。”，交谈间，闪电已经将断裂的琴头给砍成两半，与此同时还丢了个信封给苏屿川。

苏屿川轻轻一掂便知道这里面装的是现金。

“哦..哦，好的。”

事到如今，秧秧的吉他已经落到闪电的手里了，碍于他的淫威苏屿川只好照做。

——赤岭镇古城村09号

不一会儿，苏屿川便带着闪电手写的地址走出了店门。

“苏苏酱，水。”，林穗安正好买完水回来，在门口和苏屿川打了个照面。

依旧柠檬茶。

“走吧秧秧，下一站。”，苏屿川熟练的接过后喝了一口。

“去哪儿？”，林穗安歪了歪头。

“下乡。”
