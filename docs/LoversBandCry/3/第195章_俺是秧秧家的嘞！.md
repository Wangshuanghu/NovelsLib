**第195章 俺是秧秧家的嘞！**

导航指示，前往赤岭镇需要坐城际公交，车程有一个半小时。

在大城市住久后，地铁和高铁已经成了苏屿川生活的常态，他都记不清自己有多久没坐过公交车了。

林穗安也是一样，处处流露着新奇，只能傻了吧唧的跟在苏屿川身后，苏苏酱走到哪她就跟到哪，让苏屿川一度怀疑自己把她卖了她都会帮自己数钱。

公交车给人的感觉和轨道交通不同，稳健又端庄，男司机和女票务是这样一辆公交车的标配，大多还是夫妻档。

苏屿川并没有在车上找到收款码，跟着教程摸索半天才解锁洛城公交乘车码，林穗安则是有样学样，一步一步的跟着苏苏酱操作。

车上乘客的穿着大都朴素又松垮，多是上了年纪的大爷大妈，皮肤呈现被太阳晒过的古铜色，慈祥的神情给人的感觉就像这辆车一样沉稳。

驶出城市后，车内逐渐颠簸，目光透过斑驳的车窗能远眺一望无际原野，最令苏屿川动容的便是连绵不绝的农田，或者说是一片“绿色的汪洋”。

他知道我国有四大地形区，知道“高原”“平原”“丘陵”“盆地”的概念，但生活在庐州的他还是第一次真切的感受到大地彻底将它“削平高处，填补低处”的锋芒展现。

此刻的美景是说不出的，尽情用眼睛感受就好。

生活不止眼前的苟且，还有诗和远方。

“苏苏酱，这里好漂亮啊。”，林穗安陶醉道，“我决定了，我以后不想努力了就来这里隐居，好好的享受生活。”

“行，我陪你一起。”

在公交车上的一个半小时，他们舍不得闭眼，沉浸式，全身心的感受着旅途上的美景。

..

不一会儿，苏屿川就透过前窗看到了乡道旁矮矮的站台，其上赫然写着“赤岭镇”三个字。

“师傅！赤岭镇下！”，就在苏屿川准备鼓起勇气招呼时，车上一位大妈率先叫喊道。

苏屿川眼睛发亮——这不正是他们要去的地方嘛！

多亏了大妈，替他们省去了语言不通的麻烦。

“欸！中嘞！”，话落，司机师傅缓踩刹车，稳稳的将车停到了路边。

苏屿川随着人流从后门下车，林穗安小尾巴似的跟在他身后。

站台旁并没有阴凉处，正午的阳光肆虐，紫外线也很狂暴，二人都乖乖的戴上了遮阳帽。

赤岭镇在这边算是一个大镇，不少村庄都聚集在那里，因此在这站下车的人并不少。

站点靠近公路，离内部点状分布的居住区还有一段距离，走了几位后，留在这里的大都是乡里乡亲，大家熟络的唠着嗑，嘴里说着比洛城的方言还晦涩难懂的乡音。

林穗安和苏屿川不断的刷着导航，两副生面孔在他们当中尤为扎眼。

就像是外国人进了我国一样，很快，一道又一道的目光朝二人投来。

“妮儿，你是哪家的啊？”，一位自来熟的大妈朝林穗安问道。

“婶儿，她是俺家的哩。”，苏屿川凭借记忆模仿起了洛城的方言。

一声变二声，二声变四声，三声变一声，四声不变，遇到大妈喊婶子，遇到大叔喊伯伯。

他本意就是开个玩笑，离谱的是那位大妈竟真的听懂了。

“那妮儿是你家的啊？那你是哪家的？”，大妈依旧不依不饶。

“...俺，俺是秧秧家的嘞！”，苏屿川尴尬的摸起了脑袋。

闻言，林穗安幽怨的看向了他。

太能扯谎了吧？！

“哦～秧秧家的啊！小伙子长真快，俺都不认得了。”，大妈热情的拍了拍苏屿川的肩膀，语气客气得不得了。

这之后，苏屿川和林穗安就跟在她的身后往村里走。

被太阳烤干的泥巴路踩上去硬邦邦的，地面翻涌着热浪，虽然气温炎热，但这一路上苏屿川还是能看到不少戴着草帽，扛着农具的农民在地里忙活。

他们是这片大地最忠实的子民，日出而作日入而息，周而复始。

“这里不会真有一位叫秧秧的人吧？”，林穗安走到苏屿川的身边，忧心忡忡的拽住了他的胳膊问道。

前方这位大妈如果真要带他们去见那位“秧秧”，那林穗安也太尴尬了。

“不会的。”，苏屿川笑着摇了摇头，“就算我刚才没说我是秧秧家的，说我是「妄伽棠」家的或是「余观棋」家的，大妈依旧会认识。”

“这有什么说法吗？”，林穗安感到不可思议。

“没有什么说法。”，苏屿川波澜不惊道，“庄稼人都很朴实，他们不会觉得你在骗人，只会以为自己把那家人给忘了。”

“搜得死内（原来如此）！”，林穗安恍然大悟，旋即恶狠狠的掐了掐苏屿川，“苏苏酱你咋恁坏哩？”

“男人不坏女人不爱嘛。”，苏屿川嘿嘿笑道。

...

又往里走了一会儿，村落接连浮现。

这里的房屋规格很统一，方位都是坐南朝北，每家每户门前的空地上都栽了两棵对称的树。

墙的高处刷了同色调的白漆，低处则是上了层浅蓝色的墙粉。

窗户是很有年代感的长方形双格窗，正向两边开着给室内通风，玻璃窗上还有雕花。

“小伙子，你们是哪个村的啊？”，走在前面的大妈越想越不对劲，最后实在是没忍住问了起来。

“古城村。”，苏屿川平淡的回答道。

“嗷嗷！那就是俺的村子嘞！俺怎么不记得有户叫秧秧的哩？”，大妈面露怀疑。

闻言，林穗安吓了一大跳，生怕下一秒就露馅，但苏屿川这个老流氓却是无事发生。

“俺家隔壁那户叫林穗安，林穗安，婶子你认得不？”

大妈脑袋又宕机了一会儿，但很快就把自己哄好了。

“老林家啊！你讲俺晓得了！”

“对对对，老林家，老林家。”，苏屿川用力的憋着笑。

林穗安在一旁郁闷——可怜的大妈真是被这个大骗子耍得团团转呢！

既然能和大妈同路，倒是为苏屿川省去了不少麻烦，大妈走哪他跟哪，一路尾随进了村。

“婶子，俺归家去了啊！”，苏屿川告别了热心肠的大妈后，便带着林穗安顺着门牌号挨家挨户的找了起来。

“苏苏酱很擅长骗人吗？”，林穗安拽着他的手，语气幽幽道。

“哪有，少污蔑好人。”，苏屿川不认账。

“老实说，你有没有骗过秧秧？”，

“从来没有。”

...

古城村在赤岭镇里并不算小，幸好门牌的分布很有规律，最外面是一号，越往里数字越大。

很快他们就到了第九座房屋的门前。

两侧长满爬墙虎，门是打开的，整体风格与前面的房子大相径庭，唯一突兀的就是红木门上贴着的对联了。

【一生俭朴留典范】

【半世勤劳传家风】

黄底黑字，无比扎眼。

“到了。”，苏屿川面色凝重道。

对联和压岁钱是除夕的标配，但从小到大，苏屿川家一直是贴的红色对联。

他还记得小学去亲戚家拜年时，看到门口的黄色对联觉得非常帅，回去就想让苏承荣也把家里的对联换成黄的，结果换来了一顿劈。

自此，苏屿川知道了这是家里有人过世才会贴的挽联。

第一年是黄色，第二年是绿色，第三年方能换回红色。

门上贴着黄色对联，说明这家有人在不久前离世了。

“...”，林穗安站在苏屿川身旁，看着对联愣神片刻后也慢慢的沉默了下来。
