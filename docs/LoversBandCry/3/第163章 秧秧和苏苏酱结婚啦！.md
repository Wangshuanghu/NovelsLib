**第163章 秧秧和苏苏酱结婚啦！**

等店主将“痛吉他”加工完成后，林穗安和苏屿川已经带着祁苏苏将整个店从前到后逛了个遍。

祁苏苏兴致很高，此趟买了不少的拨片和效果器，苏屿川虽也心动，但毕竟不缺配件，终究是忍住了野性消费的欲望。

好不容易在高考后发了笔小财，还是能省则省吧，万一以后遇到啥突发情况也能有底气面对。

...

在挑琴弦的时候，苏屿川脑子都大了一圈。

他实在没想到小小的琴弦竟然也有那么多的讲究——琴弦的材质种类繁多，不同的材质弹出来的音色也大不相同。

最后在林穗安的建议下，苏屿川购买了“伊利克斯16052琴弦”，磷铜材质，张力大，手感偏硬，优点是音色比较脆，适合摇滚风格。

做戏要做全套，虽然只断了一弦，但苏屿川还是将一弦到六弦全部更换了一遍。

“还我妈生软弦！这也太硌手了吧？”，试弹几下后，苏屿川当即就不乐意了。

这弹起来远没有之前的舒服啊！虽然音色确实通透了些，但痛感也更通透了。

换作一个月前倒还好，但隔这么久没碰吉他，苏屿川修为尽散不说，指茧也慢慢的褪了下去，应了那句“雄关漫道真如铁，而今迈步从头越。”

“疼你就受着！波奇酱用的就是这个琴弦，人家女孩子都没说什么，苏苏酱也别太矫情。”，林穗安白了他一眼。

“后藤一里？你怎么知道她用的是这个琴弦？”，苏屿川不解道。

“哼哼，在结束乐队的《吉他与孤独与蓝色星球》这首歌里写到的，有一句歌词是「换上伊利克斯吉他弦演奏，也依旧不如人意」。”

“记这么清楚？”，苏屿川这会儿算是彻底认可了林穗安的纯度，“不过歌词不也说了「也依旧不如人意」嘛？干嘛还要给我买？”

一来二去林穗安也被问烦了，眼神幽怨的看着苏屿川嘀咕道：“苏苏酱跟个小姑娘似的，我弹木吉他都没说什么，要不你试试我的琴弦呗？”

木吉他对手指的伤害要比电吉他大得多，在林穗安看来，苏屿川这就是得了便宜还卖乖。

“不要。”，苏屿川识趣的闭上了嘴巴。

一旁，祁苏苏歪着小脑袋，饶有兴致的看着小哥哥和穗穗姐拌嘴。

她不懂什么是恋爱，但这会儿似乎渐渐明白了些——原来恋爱就是说很多很多话，吵很多很多架，但彼此又都不会因此生气，反倒乐在其中。

...

虽然嘴上喋喋不休，但看着吉他上的琴弦重新变得闪亮，苏屿川的心里还是很高兴的。

“帅哥，二到六弦还是完好的，我帮你拉出来整理好了，断了的那根一弦还要么？”，店主指了指先前卸下来的一堆品丝问道。

“不..”

“要的！”

苏屿川刚想拒绝，谁知林穗安却强硬的将那根断弦留了下来。

..

“你干嘛拿着根断弦当宝贝啊。”，告别祁苏苏后，两人走出了Guitar House，苏屿川忍不住朝林穗安问道。

“这可是苏苏酱断的第一根弦，很有纪念意义呢。”，林穗安如获至宝，满脸都是喜悦，“我俩一人一半吧？”

“随你。”，苏屿川的心跳加快了几分，但依旧保持着他的傲娇。

“好呀。”，闻言，林穗安拽住了苏屿川的右手，将一截断弦缠绕在了他的无名指上。

就在苏屿川疑惑之际，林穗安又将另一截断弦缠在了她自己的无名指上。

“原来这就是「一人一半」的含义啊..”，看着右手无名指上的琴弦，苏屿川不禁想到了新婚夫妻交换戒指。

“对呀，这可是独属于我们吉他手的浪漫。”，林穗安将她的手掌与苏屿川的凑在一起，有意无意的进行着比对，“苏苏酱的手掌好大啊，难怪学吉他进步这么快。”

“倒不如说是你的手太小了。”，苏屿川也识趣的将手掌摊开，任由林穗安打量。

“我手掌也不算小好嘛？”，林穗安不乐意了，说“手掌小”可是对吉他手最大的侮辱。

“确实。”，苏屿川思虑片刻后认可的点了点头。

林穗安的手在女生中算是很修长和白皙的了，骨型和关节也生得好看，只有在和自己比较时才会显小。

“苏苏酱，拍张照片吧。”，看着两人越发贴近的手掌，林穗安心血来潮道。

“拍啥？”

“手，我俩凑一起。”

...

林穗安又得到了一张令她满意的照片。

两人的无名指上缠绕着琴弦，又都是来自同一根，这和婚戒有什么区别嘛？

四舍五入——秧秧已经和苏苏酱结婚啦！

但没多久，林穗安又觉得这个想法好幼稚。

每次和苏苏酱待一起，自己都会变成小孩子思想，动漫说的是对的——恋爱了的少女智商为零啊。

“苏苏酱暑假有什么安排嘛？”，临近分别时，林穗安问道。

苏屿川摇了摇头：“还没想好，明明高考前列了一栏的计划表，旅游啊，报班啊，学车啊...一大堆，但真等到高考结束又不知道该做什么了。”

“嘿嘿，俺也一样。”，林穗安赞同的点了点头。

这就是「战争后遗症」——突然从紧绷的学习状态中解放后，对久违的自由感到无可适从。

也被称为幸福的苦恼，空虚又迷茫。

“班长要报吉他班吗？咱俩可以一起。”，苏屿川问道。

“才不要。”，林穗安很果断的拒绝了，“报班上课最没意思了，我还是喜欢自己练。”

“好吧。”，苏屿川有些受挫。

如果是这样，那他暑假见到林穗安的机会可不多了。

这样还不如滚回学校上课呢，至少每天都能看见她。

“苏苏酱也别报班了吧，你还有我呀。”，林穗安歪歪脑袋，笑吟吟的看着他道：“言传口教的林老师，暑假可以给你加课哦...虽然你已经学得很出色就是了。”

“好。”，苏屿川也笑了，“不管苏苏酱学得多出色，那也永远都是林老师的学生，请务必继续指导我，我太想进步了。”

“这还差不多。”

最后，两人确立了新的计划——每天下午的两点到六点，林穗安到苏屿川家来，两人再一起练习吉他。

原因有二：一是暑期林穗安的妈妈没有排班，林穗安家不方便再去；二是妄伽棠搬回了自己家住，龚玥和苏承荣又都在沪城生活，苏屿川家里没人。

“话说，苏苏酱你也太绝情了吧？刚高考完就将小妄扫地出门了？”，林穗安在得知妄伽棠搬回家后有些不解道。

苏屿川叹了口气：“唉，我为了留住sugar都差点把他裤子拽掉了，可那家伙执意要回家，还一心要打暑假工，反倒更像是我被扫地出门了..”

“这样啊...”，得知实情后，林穗安不禁伤感起来，“小妄高考完又只有孤零零的一个人了。”

没有亲人为他庆祝，也没有愉快的假期旅行，这个暑假对他来说未免太过漫长。

“这点倒是不用担心啦。”，苏屿川见林穗安又开始多愁善感，当即安慰道：“沈音符得知sugar要打暑假工后就决定陪他一起了，有她在sugar肯定emo不起来的。”

“那就好。”，林穗安松了一口气，旋即又问道：“话说，苏苏酱，他俩感情发展的咋样啦？”

苏屿川面露思索。

在五一假期，也就是沈音符表白完之后，虽然两人平日里的交流少了不少，但彼此都对这份感情心知肚明，像是小情侣在闹别扭，再说糙一点就是“调情”了。

在苏屿川看来，他俩在一起就是早晚的事，仅仅只差一个「契机」——一个让妄伽棠幡然醒悟他再这样拖下去，一切都将变得不可收拾的「契机」。

也就是要让妄伽棠明白沈音符对他的重要性，从而警示他要把握住当下。

思绪回归后，苏屿川还要回答林穗安的问题。

“跟之前的江露白和余观棋一样，就等干柴遇上烈火了。”，苏屿川耸了耸肩。

“什么叫「干柴遇上烈火」啊？苏苏酱说话也太粗鲁了。”，林穗安不满的教训道：“你应该说还差一个「特殊事件」才对。”

“别拿你galgame里面的那一套来说事可以嘛？”，苏屿川无奈道。

“嘁。”，林穗安嘟囔了一声，“galgame怎么你了...”
