**第173章 看你的了，我的主唱小姐。**

晚上九点，音乐节正式开始。

负责主持本次活动的是47cc的主唱苏在，或许是见惯了这类大场面，他表现得很是从容。

“那个，今天的音乐节是我们挨千刀的队长要求举办的，让本就不富裕的47cc雪上加霜了啊！在此我点名批评叶初阳！”，苏在说话很有趣，一下子就带动了场上的气氛。

“当然，能看到这么多的年轻人踊跃报名参演我也很激动，就像是看到了当年的47cc一样，虽然我们现在也不老就是了...哈哈。”

评委席上，47cc的另外三名队员和四位家属皆是笑骂一通——苏在真是名副其实的活宝啊。

“今天我们也不为别的，主要就是想看一下新生代的力量，这时候就有人要问了：苏在苏在，拿到名次有没有奖品啊？我的回答是：有的兄弟有的，只要上台演出就会有奖品，前三名更是有特殊奖励。”

“那我也不多废话了，让我们有请第一组乐队上台！他们是——来自亭城的「不知道取什么名字乐队」！带来的曲目是我们伟大的队长为他的女朋友余音写的小情歌，《傲娇鬼》！”

...

这里充斥着狂热的气氛。

此刻，无论是候场乐队还是现场观众，哪怕是坐在最前面的评委，都在升腾的音符中开始躁动。

炫彩的灯光，爆燃的节拍，摇摆的人群...心潮澎湃，热血沸腾。

这是来自音乐的教化，亦是来自摇滚的洗礼。

“苏苏酱..好酷啊！”

林穗安这还是第一次看现场live，从表演者变成了观众，她能更明显的感受到那股悸动。

“是啊。”，苏屿川轻笑了一声，“待会看你的了，我的主唱小姐。”

“嗯嗯，跟随我的伴奏前进吧，我的苏苏酱！”

二人说的并不多，简单交涉几句后便继续欣赏音乐，但他们又好像什么都说了。
