**第169章 安安，你还不承认！**

再次走进班级，两人都有种既熟悉又陌生的感觉。

同学们依旧说说笑笑，只不过不穿校服后变得比平时不好辨认，男生女生分成了比在校时更加森严的小团体，彼此扎着堆，像苏屿川和林穗安这样的异性组合可谓是少之又少。

...

“哈哈哈，在班级打游戏感觉真是不一样，心跳都加快了。”

“你今天穿得真好看！来和我拍照张吧？”

“姐妹姐妹...你大学报的哪里啊？”

....

释放天性后，大家畅所欲言，气氛极度活跃。

“真好啊。”，看着这曾以为束缚住了自己的囚笼，苏屿川莫名的有些眷恋。

或许今日一别，有些同学终生再难相见了。

午后的太阳很高，哪怕是把教室空调温度降到最低也无济于事，大家没一会儿就大汗淋漓，却迟迟不见班主任的身影。

在苏屿川的印象里，班主任是很严谨守时的人，上他的课也总让人正襟危坐，刻板顽固，一次都没有迟到或者缺席过，今天确实有些反常。

但班主任也没让同学们等太久。

“哈哈哈，大家都来啦，我刚去机房帮大家开空调，耽搁了点时间。”

看着小步小步跑来的老班，苏屿川觉得分外陌生。

像是一位不苟言笑的长辈在一夜之间变得亲切体贴。

原本还众说纷纭的同学们在看到班主任的那一刻皆是变得乖巧起来，玩手机的几位更是下意识的把手机揣进了兜里。

哪怕是毕业了，这份血脉压制也没有减弱分毫。

“你们啊你们，几天不见头发烫的烫染的染，真不像话，班长，清点一下人数，大家站一排！”，班主任看着穿的花花绿绿的众人不禁笑骂，随即挥挥手示意林穗安来数一下到场人数。

林穗安深知这是她最后一次履行班长的职责，从前往后数得尤其认真。

“昕念..17。”

....

“苏苏酱..29。”

清点到苏屿川时，林穗安还故意用食指在他的鼻尖碰了碰。

36人，不多不少。

汇报过后，大家便排好队前往了机房。

老班没有骗人，一进机房大家就感受到了凉意，与蒸笼似的户外截然不同。

不得不说，细节这一块还是被班主任拿捏的死死的。

按序号坐在电脑前，大家先统一上交了身份证，随后拿着准考证按流程进行志愿填报。

预填表是在家就做好了的，这会儿直接导入就行。

苏屿川从前往后又检查了一遍，确认了第一志愿是东宁大学的计算机之后便一鼓作气的点了提交。

邻座的陈佳佳有意无意的瞄了苏屿川的屏幕一眼，默默地给自己的填报表更换了顺序。

原本在第三志愿的东宁大学被她提到了最前。

她的眼睛闪过了几分迷茫，但思索片刻后还是毫不犹豫的选择了提交。

她想遵从自己的内心，她不想离自己喜欢的人太远。

走出机房后，大家都是一副如释重负的模样——接下来要做的只是静候佳音了。

苏屿川班级的志愿填报过后是隔壁的妄伽棠班，回来的路上二人还打了个照面。

“其他班还要排队填报，这会儿没啥事了，大家自由活动吧！晚上六点在大礼堂集合参加毕业典礼。”，回到班级后，班主任宣布解散。

话虽这么说，但更多的同学还是选择了留在班级，一是最后与朋友们交流交流感情，二是外面实在是太热了。

下午两三点正是一天中的最高温，此刻地面上都涌起一股无形的热浪。

这会儿，班级内依旧热闹十足。

江露白正和余观棋你侬我侬的坐在一起刷视频，这会儿他们已经没有什么要注意的了，旁若无人的秀起恩爱。

许昕念和林穗安也挤在一块，像是在说什么悄悄话，见苏屿川有意凑近，许昕念还露出了嫌弃的表情。

“姓苏的别偷听！恬不知耻！”，许昕念打发乞丐似的挥了挥手。

“不听就不听。”，苏屿川漫不经心，默默的离远了些，“你们要喝水吗？我准备出去买。”

“要的要的，快去快去。”，许昕念和林穗安正聊得火热呢，实在不想与他纠缠。

苏屿川无奈的叹了口气后便走出了教室。

他深知，要想泡到班长，必须要过许昕念这一关，偏偏这一关最难。

见苏屿川走远，二女这才敢继续聊先前的话题。

“我都找出这么多证据了，安安，你还不承认？你还说你不喜欢姓苏的？刚才你都不敢看他呢。”，许昕念笑着打趣道。

需要刻意回避苏屿川的话题自然是女生爱聊的情感话题。

作为唯一一位统观大局的人，许昕念知道安安喜欢姓苏的，也知道姓苏的喜欢安安，但偏偏两不告诉。

就是玩，就是要让他们互相拉扯。

谈恋爱哪有看别人谈恋爱有意思？

看别人谈恋爱哪有插手别人谈恋爱有意思？

“...”，林穗安把脸埋在桌子里，说什么也不肯抬头，“昕念你别说了..”

“欸，安安的脑袋在冒粉红色泡泡啊...”，许昕念觉得有趣的同时也感到欣慰——养了这么久的白菜终究还是要被猪拱。

学校东南方有个小门，小门路径两侧是永远长不大的小树，它们总是春天吐芽，夏天长叶，秋冬一来临又变得光秃秃的。

门外可以直通周边的店铺，「次元书屋」也在这一片。

路过书店门口的时候，苏屿川下意识朝里面看了眼，这会儿店内只有员工在清货，老板娘依旧在当甩手掌柜。

书店旁边就是便利店了，便利店的老板是位和蔼的阿姨，因为柜台上长期趴着一只爱睡懒觉的泰迪犬，所以这家店也让人印象深刻。

...

虽然二女没有具体说要喝什么，但苏屿川心里还是有打算的。

林穗安爱喝柠檬茶，许昕念的话..一块钱的矿泉水就行。

“柠檬茶柠檬茶，天天就知道喝柠檬茶，不知道班长在酸什么...”

苏屿川一边抱怨一边顺着货架慢慢摸索，刚准备拿最后一瓶柠檬茶时却被另一只手捷足先登。

“苏..苏同学？”

陈佳佳有些诧异，她没想到买瓶水还能偶遇苏屿川。

“学委下午好啊。”，眼见是同班同学，苏屿川也没了顾忌，热情的朝她问候，“听说你这次考了班级第一，恭喜啊，准备上哪所大学？”

她高考发挥的是全班最好的，总分680，已经赶超了林穗安。

“谢谢。”，陈佳佳轻轻点头致谢，又补充说道：“我报的东大。”

闻言，苏屿川有些诧异。

其实对陈佳佳这个成绩来说，东宁大学算不上是最优解。

但他还是选择尊重并祝福：“挺好的，我和班长也报的东大。”

“哈哈..那我们还能做同学呢。”

陈佳佳把手上的柠檬茶放回了货架上，语气有些苦涩道。

她原以为上大学就有机会了，没想到还有林穗安这座大山在啊...

“你不要了？”，苏屿川见陈佳佳把柠檬茶放回去有些迟疑道。

“嗯，苏同学你拿吧。”，陈佳佳说罢重新拿了一瓶矿泉水。

“谢谢。”，苏屿川毫不客气的接手了柠檬茶。

“你买这么多呀？”，见苏屿川一共拿了三瓶水捧在胸前，陈佳佳忍不住问道。

“我这是为乐队的主唱小姐和运营小姐打工呢。”，苏屿川半开玩笑道。

陈佳佳的眼神黯淡了几分，但还是故作平淡的回应：“StarryStar的文化节演出很赞呢，我已经关注你们的视频号了，继续加油啊。”

“感谢学委支持。”，苏屿川笑着点了点头。

...

“苏同学，冒昧的问一句，你是不是喜欢林班长？”

结完账走出店后，陈佳佳停住了脚步，再也忍不住问道。

“嗯。”

苏屿川回答的毫不犹豫。

“非常喜欢。”

他没有隐瞒什么，渐渐的，他也不想再当那个缩头乌龟。

虽然还没有勇气对当事人说，但循序渐进的过程还是要有的。

“我...祝你前程似锦。”，陈佳佳想说的话挂在嘴边，终究还是没能说出口。

“谢谢，你也。”，苏屿川礼貌的示以微笑，旋即转过身头也不回的走了。

“我喜欢你啊。”

陈佳佳看着暗恋三年的男生的背影，眼睛里的遗憾和落寞终于倾吐而出。

学习上，林穗安是我不可逾越的大山。

哪怕是在感情上，她也是我不可逾越的大山。

...

看着手上未开封的矿泉水，陈佳佳再没了要喝的欲望。

她知道青春总是伴随着伤痛和遗憾。

但这也太讽刺了些。
