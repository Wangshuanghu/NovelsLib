**第181章 殉情吧！苏苏酱！**

“苏苏酱会不会也很眼红呢？”，想到这，林穗安悄悄的瞄了眼苏屿川。

苏苏酱已经没在看小妄和小沈了，而是翘着个二郎腿散漫的坐着，手上正拿着手机朝舞台上拍照。

林穗安好奇的探过头去，却见苏屿川正在放大屏幕看美女！

“不知廉耻！痴汉！臭不要脸！”，林穗安的小宇宙爆发了，狠狠的掐住了苏屿川的脖子。

她还是头一回吃这么大的醋。

殉情吧！苏苏酱！朝三暮四朝秦暮楚二三其德的渣男！

“秧秧你干嘛啊？！”，苏屿川吓了一跳，两腿乱蹬，直直摔到了地上。

压在他身上的林穗安也没好到哪去，失力后整个身体都贴上了苏屿川。

“你好冒昧啊。”，苏屿川不满的嘟囔道。

他有种被强推的感觉。

“你..你是不是在偷拍美女！！”，林穗安一点也不理亏，一边捏着他的脸一边质问道。

“哈？”，苏屿川苦不堪言，“我只想看一下她用的是什么吉他啊！”

“..”

这下轮到林穗安尴尬了，她这会儿还压在苏屿川身上呢！

“对不起，错怪你了。”，林穗安轻轻撇了撇嘴巴。

....

此时演出已经进行到了第20组，离本次活动结束也不远了，值得一提的是在StarryStar之后并没有再出现65分以上的存在，他们依旧是稳居第二。

最后的压轴乐队的演出也只能算是差强人意。

“OK啊，很感谢各位带来的表演，看得我是心潮澎湃，马上就要十二点了，现在请允许我们47cc的这群老人来凑个热闹！”，苏在接过麦克风后说话语气显然激动了不少。

没有一个乐队是不渴望舞台的，终于轮到他们登场了！

虽是深夜，场下的欢呼声却经久不衰，甚至达到了本次音乐节的最高峰。

比起名不见经传的21支学生乐队，还是47cc的吸引力更大，要知道，近3000名观众几乎都是奔着47cc来的啊！

“当然，我们这群老人凑完热闹之后就是喜闻乐见的颁奖环节了，21支乐队的小朋友们稍安勿躁。”

“今天我们带来的新歌是——《七月》！在场的各位都是第一见证人！”

...

主唱苏在，吉他手叶初阳，鼓手谢不伪，贝斯手钟星回...

苏屿川永远也忘不了这一幕。

他从没有看过演唱会，也不知舞台上的氛围应该如何，但在47cc的表演中，他能感觉到身上的每一个细胞都在悦动。

振聋发聩的鼓点无可挑剔。

浑厚圆润的贝斯浑然天成。

就连主唱苏在的歌声也是极具感染力，这也正是林穗安一直在尝试探索的奥义。

“感染力”——简简单单的三个字，确是无数音乐人的毕生追求。

如何引起观众的共鸣从而传达歌中的意旨？唯有清楚这一点才能深刻的领悟音乐教化作用的真谛。

而撑起这一切的，化音符为画卷的，正是47cc队长叶初阳的吉他。

一切都是那么的完美，从指法到节奏，没有一处是不合拍的。

但最厉害的是，叶初阳的琴音一点都不死板，仿佛融入了演奏者的无数遐思，又更像是春风化雨，令听众沐浴在极度灿烂的阳光下。

现场的live远比在音乐软件上听到的更震撼。

摇滚的真谛——食指和小拇指朝上的那一刻，一切烦恼都将成为过往云烟。

“这才是吉他啊！”，苏屿川被狠狠的折服了。

原创的作词作曲让人耳目一新，每一处的变调都无迹可寻，却又殊途同归。

七月的到来伴随着这样的一首歌，浪漫至极。

“真羡慕sugar啊，此情此景还收获小女友一枚，人生赢家。”，苏屿川拍了拍妄伽棠的肩膀打趣道。

“屿川，还不是你们合伙骗我的。”，妄伽棠苦涩的笑了笑。

“有情人的事怎么能叫骗呢？真要说那也是「撮合」！”

闻言，妄伽棠陷入了短暂的沉默。

“屿川，你说我真的能对小沈负起责任吗？”

“一定能！”，苏屿川的语气不容置疑，“除你之外，换谁来都不行！”

“好，我知道了。”

妄伽棠的目光重新坚定，看向了那个满眼是他的女孩子。

“干嘛？不会是把我骗下来之后就要分手吧？”，见妄伽棠步步逼近，沈音符吓了一大跳。

她知道妄伽棠向自己表白是一时冲动之举，生怕他要反悔。

“不是。”，妄伽棠连连摆手，“我想说的是，我的心很小，只能装得下一个人，所以..你能一直陪着我吗？我发誓会对你好的。”

“吓死我了。”，沈音符惊魂未定的拍了拍胸口，旋即灿烂的笑了起来：“乐意至极，我的超人班长。”

“你的父亲教了你那么多大道理，有没有教过「一言既出驷马难追」啊？要是爽约我可是会恨死你的。”

...

画面太美，苏屿川不敢看。

欣赏演出时，大家默契的分成三组。

江露白和余观棋，妄伽棠和沈音符，许昕念和林穗安，只有他这个孤家寡人四处乱窜，像皮球似的被踢来踢去。

47cc的演出结束后，观众们皆是亢奋无比。

相较于21支学生乐队，47cc简直就是降维打击！

“录音棚版本已经在路上了，过几天就会发布，大家敬请期待吧！”，苏在一边喘着气一边激动道，“现在，进入本次音乐节的尾声！有请我们伟大的叶初阳队长为获得前三名的队伍颁奖！”

“当然，没有获得前三名队伍也不用灰心，还是那句话，只要参加就有奖！”，苏在继续补充道，“另外十八支队伍可以凭借排名领取相应数量的纸巾，分着用，找个地方好好擦眼泪。”

“哈哈，开玩笑的，那十八支队伍可以来评委席挑选47cc的亲签专辑哦，虽然不是很值钱，但还是有纪念意义的..或许。”

...

亲签专辑这种东西，对喜欢的人来说是无价之宝，对不喜欢的人来说就是路边一条，但在场的无疑都是摇滚爱好者，听闻有47cc的专辑又怎能不激动呢？

“4到21名的奖励都这么好？那前三名是得有多逆天啊？”

观众们不禁议论纷纷。

“最后，有请本次音乐节获得前三名的乐队的三位队长上台！洛城，「ToNight」乐队的闪烁，庐州，「StarryStar」的苏屿川，江宁，「GTrun」的陆秋秋。”

...

站上台后，苏屿川忍不住打量起了叶初阳。

离得近了才发现他长得确实很帅，难怪能靠颜值为47cc圈粉。

但除了帅气，他身上那股难言的叛逆的气质也让苏屿川倍感尊敬。

听说叶初阳的学历很高，现在正在上大三，能将乐队和学业做到如此平衡不可谓不强大。

“喂，信球货，林穗安的吉他是不是摔坏了？”，47cc的众人在清点奖品的时候，闪烁再次找到了苏屿川搭话。

“咋？你要送秧秧一个？”，苏屿川对她的好感度并不高，回答的也很漫不经心，“不过我事先说好，就算你送了她也不会加入你们乐队的。”

“我才不送呢，要送也应该你这个当男朋友送。”，闪烁连连摇头，“我刚才看了，摔得挺严重的，一般琴行肯定修不了，不介意的话可以来我家店里修，我老爹可是专业的！”

苏屿川默默低下了头——先前整理残骸时他们就发现了，林穗安吉他的琴板和琴弦已经悉数崩断，琴头裂开后弦距也高了好几厘米，就连她本人都没有要修理的打算。

但那是秧秧的第一把吉他，苏屿川不想就这么放弃。

“你不会漫天要价吧？”，苏屿川试探道。

闪烁摇了摇头，淡淡开口道：“不会，友情价，就当交个朋友，你加我好友，我把地址发给你。”

闪烁这意思是要他们自己跑一趟。

“你不回洛城吗？帮我带着呗？”，苏屿川疑惑道。

“我们ToNight难得出趟远门，想在长三角这边多玩玩，半个月内应该不会回洛城，去的话跟我说一声，我跟老爹打个招呼。”

“好吧。”，苏屿川点了点头，“多谢了。”

“没事，毕竟我挖了你的墙角嘛，就当是给你的补偿。”

看着闪烁发来的地址，苏屿川陷入了思索。
