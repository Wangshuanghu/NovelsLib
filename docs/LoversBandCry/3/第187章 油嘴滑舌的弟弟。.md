**第187章 油嘴滑舌的弟弟。**

花花被苏观铃带去了沪城，sugar也从家中搬走，这使得苏屿川睡起觉再没了干扰。

他理应睡得天昏地暗，但次日清晨，却是早早的起床了。

三天挣大钱，时间紧任务重，苏屿川一点都不敢懈怠。

按照余观棋的建议，他乘着地铁抵达了三里庵那边。

七月的清晨已经算不上温度适宜了，仅仅只是走几步就让人感到浑身燥热。

“如果不是要向秧秧表白，这会儿我应该还在家里吹空调玩电脑吧？”，苏屿川自嘲道，“往年的暑假貌似都是这么过的..现在想想还真是颓废..”

...

买了份手抓饼，一边吃一边漫步在街道上，脑子里无数的想法逐渐跳了出来。

卖小吃？

..算了，会不会变成致命毒师尚且不论，光是天天早起和城管玩捉迷藏就很折磨人了。

跑外卖？

..庐州市中心虽然接单很方便，不过交通状况还是太复杂了些，危险系数高尚且不说，自己还做不到老马识途。

..

苏屿川想了很多种可能性，但却始终拿不定主意。

人才市场那里倒是有挺多人的，反正闲着也是闲着，赶到后苏屿川在招聘栏上一字一字阅览着。

弹性工作时间？意思就是要加班呗？

工资2000-5000？看来是只有2000了。

包吃包住？直接说全天无休就是了。

越看他眉头皱的越紧。

不远处有个戴墨镜的猥琐男看苏屿川面露愁容，满脸堆笑的凑了过来。

“小兄弟，我现在有份躺着挣钱的工作，你只需要买张机票就行了，当然，还要支付我的一点微不足道的中介费。”

苏屿川本就烦的一批，听到这话后更是气得一脚踢到他的屁股上，将猥琐男踹翻在地。

“滚啊！你他妈嘎你自己腰子去！”

此地不宜久留，真是什么人都有。

离开了之后，苏屿川有些忧心忡忡。

如果今天来这里的是林穗安的话，她会不会真的被骗呢？

他开始幻想起了那个场面。

....

“小美女，我们有份躺着赚钱的工作...”

“你刚刚说什么？”

“我们有份躺着赚钱...”

“上一句！”

“小美女？”

“嘿嘿，嘿嘿嘿....带秧秧去吧。”

那应该会很有趣吧？

苏屿川忍不住笑了起来。

一想到林穗安，他心里的郁闷都少了不少。

为她，再苦再累都是值得的。

...

但兜兜转转一上午，苏屿川还是一无所获。

暑假工本就不受待见，更何况他还是只打三天工的临时工，没有一个工作岗位需要这样的新兵蛋子。

如此看来，也只有走日结这一条路了，偏偏日结的工资又低的可怜。

“算了，有钱赚总比没钱赚好。”，苏屿川叹了口气后还是拨通了余观棋的电话。

“小样，我老实了，让我去发传单吧。”

...

正午，淮海路

“...总而言之就是这些流程，小同志要认真干啊，我们讲究多劳多得的。”

乐器培训中心的老板一五一十的向苏屿川介绍了工作流程，一口流利的官腔听的苏屿川差点睡着。

总结下来无非三点。

不怕拒绝的决心，不怕制裁的勇气，不怕人群的白眼。

简称脸皮厚。

这对皮实的苏屿川来说自然是没有任何难度的。

“苏嘉豪你加油，我会心疼你的。”，培训班楼下，余观棋看着苏屿川手上厚厚的一沓传单不禁汗颜。

大太阳底下发传单绝对算不上美差，苏屿川这也太苦逼了。

“真心疼我的话就帮我发了。”，苏屿川白了他一眼。

“滚一边去，我还要上课呢，你大不了找个地方统一销毁，反正没人监督。”，余观棋压低了声音后开始出谋划策。

“算了吧，我还是有职业操守的。”，苏屿川笑骂道。

实则他有更长远的打算。

“好吧，那我上课去了。”，余观棋说罢便带着贝斯朝店里走去。

“提前祝你表白顺利，恭喜啊。”

..

他单肩背着黑色的琴包，包上零零散散挂着几个玩偶，那些都是江露白在过去送给他的，前行的步伐稳健又笃定。

此刻，余小样的身影似乎与过去的苏屿川重叠，或者说，与成千上万个开始追逐音乐梦想的少年们重叠。

苏屿川相信他一定会学有所成。

...

告别了余观棋，苏屿川也开始了他的工作。

三思而后行是他刻在骨子里的习惯，这也使得他做事从不盲目。

工作时间是下午四点到九点，如何将工作效率最大化成了他现在主要思考的问题。

发传单被拒绝是屡见不鲜的事，但任务额是固定的，与其广撒网，不如等退潮。

培训班地处闹市区，下午四点到五点，街上的人群大多是游客，这些绝不是潜在的客源。

此时如果推销说“音乐班培训了解下”，那么成效必定会大打折扣。

听说不少大学生发传单都是说“帮忙丢一下，谢谢。”，那样确实有不少人出于好心接受，苏屿川也想过要那么干，但道德还是占领了高地。

那又该如何才能让人拒绝不了一张碍眼的破纸呢？

很快他就有了想法。

人们不愿意接受传单无非是因为它无用，如果它变得有用的话，陌生人对它的好感势必会陡然增加。

七月初的庐州有着令人发指的高温，高楼林立的CBD更是暑气不散。

苏屿川不禁开始埋怨起了乐器班老板——如果增加点成本，将传单做成扇子的形式，那发起来也不至于这么麻烦。

于是苏屿川从手机上认真学习了如何用A4纸叠小扇子，出于良心，他还保留了传单中央最醒目的招生文字。

“真抠门，真抠门，真抠门！”，每叠一张扇子苏屿川都要骂一句。

大众不会乐意手中多一个无用的垃圾，却很乐意多一个有用的暂时性的小物件，哪怕它微不足道。

“小姐姐，天气热拿个小扇子呗...”

这样子推销，出发点从为自己宣传变成了为别人考虑，要知道，人都是以自我为中心的，讨厌别人的冒犯却满足于别人的关切。

哪怕下个路口他们便发觉这破纸压根扇不起风，然后无奈的将它丢进垃圾桶，但在这期间，内容宣传做到了，他们也不会感到厌烦。

因此，仅仅过去一个小时苏屿川便完成了任务量的三分之一。

傍晚五点到六点，是都市白领们下班的时间。

上班族很忙，肯定是没闲心去搭理这意义不明的乐器培训的。

苏屿川很明白这一点——CBD的社畜可是怨气最重的一群人，还是不要招惹为好。

但可以用奇兵。

晚高峰之前，趁人群还不算拥挤，他不紧不慢的走进了奶茶店。

“姐姐，我能提高你们的销量和营业额，能不能商量个事...”，苏屿川双眼清澈的问道。

“这...我们做不了主的。”，看着少年手上厚厚的传单，店员的语气无比为难。

苏屿川推销的意图简直不要太明显。

“好吧，我知道了，那我要一杯柠檬茶和两杯黑糖珍珠奶茶。”

等店员把三杯做好之后，苏屿川只接过了柠檬茶，剩下的两杯都没有拿。

“姐姐，我请你们喝。”，苏屿川笑着说道。

奶茶店兼职的是两名女大学生，看到长相如此俊俏的小弟弟顿觉心中小鹿乱撞。

但处事原则她们并没有忘。

“你的心意我们领了，可我们只是打工的...传单这事我们做不了主，奶茶你还是拿去吧。”，收银的小姐姐戴着口罩，只露出一双眼睛，语气带着歉意道。

“我知道的，刚才你们做奶茶的时候我叠了40朵小花，方便的话可以放到前台吗？装饰性质的，客人想要的话就让他们拿。”

说罢，苏屿川也没有再为难她们，留下一堆愚拙的手工品便离开了。

两名女生看着苏屿川离开的背影，嘴角忍不住上扬——做奶茶这么久，还是第一次有客人主动请员工喝的。

他为二人准备了比较贵的黑糖珍珠，却是给自己留了最便宜的柠檬茶。

拿起一朵不算精致的小纸花，她们打量了一会儿又一会儿。

“油嘴滑舌的弟弟。”

一位小姐姐说道。

“小脑袋倒是挺聪明的，知道赶在晚高峰前来，这小纸花客人随手一拿就带走了。”

另一位店员也喃喃道。

“我倒挺好奇他一开始说的办法是啥，发传单还能提高我们的销量。”

“哈哈，谁知道呢，要是老板在就好了。”
