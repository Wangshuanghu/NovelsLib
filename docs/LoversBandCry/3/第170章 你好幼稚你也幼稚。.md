**第170章 你好幼稚/你也幼稚。**

“凭什么给安安带的是柠檬茶，给我的就是矿泉水？”

见苏屿川明目张胆的玩差别对待，许昕念差点被气上天。

这还没在一起呢，姓苏的就对自己这个“媒人”如此不尊重，在一起之后怕不是要把自己当成路边一条？

“因为她比你可爱 ”，苏屿川回答的语气很是散漫，就像是在陈述一件人尽皆知的事，“而且我也不知道你喜欢喝什么。”

“嘿嘿..”，林穗安有些小开心。

不止是因为苏苏酱记住了自己的喜好，更是他对自己那毫不掩饰的偏爱。

要是和他在一起，秧秧一定会很幸福。

但这个呆瓜还是在钓着自己呢，像个不会开花的木头。

...

这会儿离毕业典礼还有一个多小时，妄伽棠和沈音符也填完志愿回来了，StarryStar的七人闲来无事又开了个小会。

讨论的重点还是两天后的江宁音乐节。

“抽签顺序和具体流程出来了吗？”，苏屿川向许昕念问道。

大家对她信赖有加，这些事都由她这个“经纪人”一手负责。

“嗯，我本来准备今晚在群里发的，挺长一大串呢，现在人都在，你们都看一下吧。”，许昕念说罢便掏出了手机。

众人凑上去认真的看了起来。

【47cc无敌炸裂的音乐狂欢】

地点：江宁玄武湖莲花广场

时间：6月30日晚上9点到12点。

参演乐队共21支，顺序全随机..

...

“原来是露天演出啊。”，江露白松了一口气，“这不就跟在淮海路那次一样嘛？”

“不一样的。”，余观棋摇了摇头，“别低估了47cc的影响力，听说等其他乐队表演结束后他们会有首新歌的live，光是这点就能吸引不少粉丝去了。”

沈音符没有着急说话，而是来来回回将注意事项看了一遍：“门票共2777张，票价7元....这也太..”

“七块？！”，江露白吓了一跳，“这是公益演出吗？怕是连场地费都赚不回来吧？”

“不是公演，但也差不多。”，许昕念咧了咧嘴巴说道，“据说收七元的门票单纯是为了迎接七月...”

“还真是有个性啊..”，林穗安对这支名为47cc的乐队也好奇了起来。

“sugar，你在想什么呢？”，苏屿川见妄伽棠一直没说话，默默的在那沉思，忍不住问道。

“我在想为什么47cc已经入不敷出了还要收门票费。”，妄伽棠淡淡回答道。

“想明白了吗？”，苏屿川觉得蛮无厘头的，笑着追问道。

“想明白了。”，妄伽棠故作高深，“因为玄武湖公园有桥，47cc要收过桥费。”

“...”

“。”

“，”

“空调温度是不是太低了..有点冷啊。”，沈音符忍不住缩了缩身体。

“sugar真是冷笑话大王呢，还有上次的大笑瑜伽也是..哈哈哈。”，苏屿川被妄伽棠的话雷得头皮发麻，仿佛下一秒就要被冻僵。

...

“昕念，我们的顺序是多少？”，回过神后，林穗安问道。

“第17组，大概是在11点到11点10分登台。”

闻言，众人皆是面露思索。

在21组当中排第17，这已经算是很靠后了，偏偏又离压轴的乐队有些距离。

观众的新鲜感到那会儿估计也消失的差不多，这个形式对他们并不利。

“其他乐队演奏的歌有透露吗？应该没有多少人选《小叶子》吧？”，苏屿川问道。

如此情况下，独树一帜便是他们最后的底牌，如果演奏的歌曲也同质化严重，那StarryStar注定只能当路人甲了。

“我看看..选得最多的是《傲娇鬼》，有足足八支乐队选那首，《小叶子》的话..有两支。”，许昕念一边翻看一边喃喃道。

“还真有人这么猎奇啊？”，余观棋嘴巴都惊得张大了。

两支乐队演奏同一首歌，孰强孰弱一听便知，这也在无形之中给他们上了压力。

“还有更糟糕的，另一支演奏《小叶子》的乐队是第16组，就在我们的前面。”，许昕念面露愁容的叹了口气。

听到这话，大家都有些忐忑。

俗话说没有对比就没有伤害，这次的演出怕是要伤痕累累啊。

“这也太棒了吧？”，众人缄默之际苏屿川站了出来，语气兴奋道：“这不就是皇城PK嘛？多有意思啊？”

“队长不愧是队长呢..”，沈音符被苏屿川的话激励到了，眼神也再次坚定，“不过队长说得对！”

气氛再次被点燃。

“兵来将挡！”，江露白伸出了小拳头。

“水来土掩！”，余观棋笑着用手掌包裹住了江露白的小拳头。

“攻无不克！”，苏屿川也顾不上中二不中二了，激动的同二人碰了拳。

“战无不胜！”，林穗安默默的将手贴在了苏屿川的边上。

“那还说啥？”，许昕念自信的露出大白牙，随即便将拳头递了上去，“开团秒跟！”

“一往无前！”，妄伽棠也被感染。

“俺也一样！”，沈音符雀跃道。

“StarryStar！Shining！”

....

这腔热血一直延续到了毕业典礼前。

但只要往大礼堂那一坐，苏屿川就一点脾气都没有了。

他上学的时候最烦的就是去开大会，往往一坐就是好几个小时，校长唠叨完主任唠叨，主任唠叨完老师唠叨，总而言之就是唠叨。

今天的毕业典礼也不遑多让。

“今年，我校高考成绩再创佳绩，六班的妄伽棠同学总成绩更是被隐藏，据透露，其总分位列全省前五，全市第一...”

果不其然，妄伽棠直接成了全场的焦点，市教育局更是协同校组织来给他颁奖学金。

总奖金50000元，外加大学四年学费生活费全包。

太权威了。

苏屿川看得心动不已。

sugar也是过上好日子了。

“这三年，你们砥砺前行；这三年，你们所向披靡，这是遍布成长与收获的三年，这是...”

主持人还在唠叨个不停，但苏屿川早已无心去听。

他百无聊赖的在礼堂的四下打量——同学们的神色都很精彩，松弛又肆意。

有种在上学时期很难见到的活人感。

看着看着，苏屿川的目光又锁定到了林穗安身上。

她就坐在自己前面，正聚精会神的捣鼓着手机。

离得很近，苏屿川能看到林穗安那是在玩视频软件的特效。

屏幕上，她的脸一会儿变成黄瓜一会儿变成小狗小猫，每当碰到抽象的，她还会“嘿嘿”的在那傻乐。

班长真好玩。

苏屿川也被她的傻里傻气感染了，目不转睛的看着她玩特效。

似乎是注意到了后方苏苏酱看傻子的目光，林穗安仔细看了眼视频，果真捕捉到了苏屿川举着的摄像头。

“苏苏酱，坏！”，林穗安暗暗笑了笑，旋即将摄像头转向了他。

这下好了，屏幕里的两个人都被上了特效。

“你好幼稚。”

“你也幼稚。”
