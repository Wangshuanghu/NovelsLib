**第197章 苏苏酱喜欢秧秧嘛？**

“苏苏酱干嘛去？”，见身旁的苏屿川坐起了身，林穗安如梦初醒。

“俺去帮奶奶烧锅，秧秧就在此地不要走动。”，苏屿川嘿嘿笑道。

“...”，林穗安哪里不知道他这是在占自己便宜呢？鼓起嘴巴白了他一眼，“哦。”

地处偏僻的农村，奶奶的家里没有电饭煲和天然气，煮饭用的还是土砌的炉灶。

炉灶上方的空间很小，灶神被安置在厨房南面，香炉旁还有腊月供奉的糕点。

浓浓的乡土气息和生活韵味扑面而来，这点是苏屿川在城市里体验不到的。

他想起了高中必读的那本费孝通的《乡土中国》，其中有这样的一段话：家必须是绵续的，不因个人的长成而分裂，不因个人的死亡而终结，于是家的性质变成了族。

“奶，俺来烧锅。”，看着奶奶在灶上忙前忙后，苏屿川心里很不是滋味。

“阿杂菜（脏），不中，俺老婆子来，你歇着嘞。”，奶奶连忙招手，说什么也不愿意让苏屿川上。

“喵事，俺想试试哩。”，苏屿川笑道。

他一点架子都没有，脸上的表情始终恬淡，对柴草间的灰尘和蚊虫毫不在意。

苏屿川此前从没有做过这些，但上手却出奇的快，点燃引火草塞进炉子，拿柴，劈柴，入炉，控火，一气呵成。

出于好奇，林穗安从里屋探出个小脑袋来偷看。

下方的苏屿川坐着烧火，上方的奶奶站着炒菜，这样的做饭方式林穗安从没见过，一切都是那么的写意清奇，袅袅炊烟从烟囱中升起，她不禁看得有些痴了。

对田园牧歌的向往是国人刻进骨子里的基因。

三人份的饭煮的很快，奶奶先是招呼二人坐下，又客气的拿起盘子去了厨房盛菜。

“来滴太突然嘞，俺老太婆也没得什么招待你们的，连点肉都来不及打。”，说罢，奶奶热情的给林穗安和苏屿川散了双筷子。

在她看来，这二位都是小孙女的朋友，虽然帮闪电做事让她很不爽，但来即是客，待客之礼不能丢。

桌子上的菜很朴素，鸡毛菜汤，炒空心菜和炒豇豆，一点荤腥没有。

“谢谢奶奶招待。”，苏屿川真诚的点了点头。

没有丝毫怨言。

要知道今天本就是他们厚着脸皮来蹭饭，能被如此热情款待，他自然是感激不尽，又何谈怪罪呢？

“谢..谢谢奶奶。”，林穗安有样学样。

乡下人生活一向节俭，面对突然到访的两人，这两菜一汤已经是极好的了。

两张长板凳，奶奶坐在主位，苏屿川二人正对着她，气氛其乐融融。

奶奶吃饭很安静，苏屿川进食时也不爱说话，倒是林穗安憋得难受，胆怯的如小兔子一般一点一点的咀嚼着白米饭，一点菜都不吃。

“妮儿，吃菜啊，你都长这么瘦了。”，奶奶说罢便给她夹起了菜。

“谢..谢谢！”，林穗安受宠若惊。

“阔气嘞（客气了）。”，奶奶乐呵呵道，“俺这有些菜，你们可能帮我带去给闪电啊？”

林穗安歪歪脑袋表示没听懂。

“好，俺知道了。”，苏屿川回以微笑。

这对母子真是一个比一个别扭。

...

午后阳光依旧明媚，闪烁奶奶躺在太师椅上，轻轻地摇着蒲扇养神，眼见碗还没来得及洗，苏屿川便在后院打了一盆井水，认认真真将碗筷洗好。

林穗安坐在门口，瞭望着一望无际的稻田，炎热的风吹起麦浪，她的内心也静谧了许多。

乡下的生活似乎也挺美好的。

老年人的瞌睡总是很多，不一会儿，奶奶手中的蒲扇就停止了摆动，见状，苏屿川静悄悄地进里屋拿出小被子，小心翼翼地搭在了她的肚子上。

“秧秧，走吧。”，苏屿川拎起奶奶事先准备好的蔬菜，轻声招呼林穗安道。

刚才拿被子时，他顺手将闪电的信封放到了奶奶的床头柜上，虽然手段卑鄙了些，但也算是圆满的完成任务了。

“哇嘎达。”，林穗安缓缓起身，望着绿油油的稻田，心中还有些意犹未尽。

...

下午两点是一天中最炎热的时候，体感估计有四十度往上，两人没再走大路，而是顺着田埂边上有阴凉处的林道走。

“感觉会随机刷出来蛇呢..”，埂上的路很窄，不支持两个人并排，林穗安轻轻的牵着苏屿川的衣角跟在他身后。

“这么热的天，蛇也不会愿意出门的。”，苏屿川虽然嘴上这么说，但心里却在不断打鼓。

从小到大，他最怕的就是蛇了。

往前走，远离了居民区，农田也越来越密集。

“苏苏酱，这田里种的是小麦还是水稻啊？”，林穗安傻傻的问道。

她很喜欢这片由农作物铺成的绿地。

“这边气温挺高的，七月应该是种晚稻。”，苏屿川思索片刻后笑了起来，“哈哈，这里有一整田的秧秧呢。”

不管是水稻还是小麦，小时候都统一叫秧苗。

林穗安的小名也不难理解，“穗”的小时候不就是“秧”嘛。

林穗安丝毫不因苏屿川打趣自己而生气，反而是饶有兴致的问道：“那苏苏酱喜欢秧秧嘛？”

“喜欢啊，没有秧秧我吃什么的。”，苏屿川回答的漫不经心，仿佛是在说一件老生常谈的事。

见苏苏酱一点都没害羞，林穗安顿感没劲。

坏，又不把秧秧当回事。

不知不觉，二人走到了河边。

“这是洛河吗？”，林穗安好奇的问道。

“嗯，是支流。”，苏屿川仔细看了看手机地图。

洛河对洛城有着非凡意义，它像一位时而严苛时而慈爱的母亲，丰饶与亢旱都与它相干。

河水款款流过，宽近十米，深达两米，是当地人赖以生存的根本。

“苏苏酱知道《洛神赋》吗？”，林穗安难掩激动，“秧秧今天又圣地巡礼啦！”

“傻瓜。”，苏屿川笑了笑，旋即便准备绕路走。

“苏苏酱！”，林穗安扯住了他，“你热不热？”

“热啊。”，苏屿川不假思索道。

“那..我们去玩水吧。”

...

苏屿川咽了口口水，惶恐的看着水面。

雨季尚未来临，河水还算澄澈，眯起眼睛仔细观察的话，还能看到水面上隔三差五冒出的鱼儿吐出的泡泡。

“我不会游泳。”，苏屿川有些不好意思道。

自小就接受的防溺水“六不一会”教育让他对陌生水域有着本能的恐惧，他深知哪怕是浅水，也有淹死人的可能，更何况这里足足有两米。

“拉着我呀。”

说罢，林穗安的手松开了苏屿川的衣角，转而拉住他的手腕。

虽然动作有些暧昧，但是林穗安没有想那么多，毕竟当务之急是保障苏苏酱的安全，可苏屿川却是老脸一红，脑袋都宕机了片刻。

由浅入深，水温越来越低，等到水快没到林穗安的胸口时，她停下了脚步。

回头看，水才淹到苏苏酱肚子上一点点，应该是安全的。

慢慢松开了苏屿川的手腕，林穗安这才后知后觉的羞涩起来。

她个子比较小，又走在前面，这会儿已经有一小部分头发被水沾湿，飘散在水面上，衬衫和裤子的布料也尽数湿透，隔着清澈的河水，影影绰绰的显现出婀娜的身姿。

虽然林穗安偏瘦，但却是选择性的，怎么说也有tan45º，只不过瞄了一眼，苏屿川就红了脸，急忙把视线挪开。

这这这...这就是戏水play吗？
