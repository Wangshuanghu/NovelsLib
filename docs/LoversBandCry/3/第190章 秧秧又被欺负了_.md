**第190章 秧秧又被欺负了..**

七月五日

苏屿川与林穗安约定在这天下午去洛城维修吉他，但事情远没有表面听起来那么简单。

...

收拾好旅行用具后，林穗安先是到了苏屿川家门口，准备和他一起去火车站。

【秧秧】：苏苏酱乖乖～把门开开。

【苏苏酱】：来了。

门打开后，两人大眼瞪小眼互相打量了一通，旋即又都害羞的收回了视线。

“苏苏酱..今天的打扮很好看呢..”，林穗安转过头，语气娇羞道。

兴许是吸取了五一假期江宁之旅的教训，苏屿川这回穿的很是得体，过眉的刘海也被撩起来，露出了前额，整个人都显得阳光。

“秧..秧秧也好看。”，听了三位军师的建议后，苏屿川也学聪明了，收起了他毒舌的坏毛病。

当然，林穗安今天的穿搭也确实没得喷。

浅蓝色的条纹衬衫，搭配上同色系的蝴蝶结领结，下身是轻飘飘的小裙子，整体风格清新又带点甜美的少女感。

头发似乎是刚洗过，看着柔软又舒适，漂亮的脸蛋不需要任何化妆品的衬托，尽显绝色。

与她一比，自己简直邋遢的像是流浪汉。

“....”

“....”

一通互夸后，两人都陷入了沉默。

他们眼前站着的都是自己喜欢的人，明确彼此的心意后，举手投足都开始变得拘谨起来，生怕一个不小心就让对方对自己幻灭。

“走，走吧。”，苏屿川罕见的结巴了起来。

“好...”，闻言，林穗安悻悻的退后一步，给苏屿川出门留以余地。

虽说在感情上亲密了不少，但他们的所有小动作都开始变得陌生，就像是在招待贵客似的。

这也是关系由“挚友”向“情侣”转变的直观体现，虽然现在还没有彻底落实，但没关系，苏屿川马上就要迈向那一步了。

...

日头很高，街道旁的绿树不断传来蝉鸣，头顶的白云一朵连接一朵，厚厚的，天则是蓝蓝的。

这便是夏天，热烈而永恒的夏天。

苏屿川撑着伞，默默走在林穗安身边，两个人靠的很近，以至于肩膀都会时不时的撞上。

“秧秧，挺热的，我来帮你背包吧。”，见林穗安背上背着琴包手上又拎着书包，苏屿川贴心的请缨道。

“不..不行！”，林穗安慌忙的摆起手，想都没想就拒绝了。

“咦？”，苏屿川倍感奇怪。

林穗安可不是会跟自己客套的女生，而且她说的是「不行」，而不是「不用」。

不对劲，十分有九分的不对劲。

“苏苏酱非要我说明白嘛？”，林穗安见他仍旧目光灼灼的盯着自己的书包，不满的鼓起了嘴巴。

“可以的话自然最好，我怕你把小音藏书包里，看我不爽就放狗咬我。”，苏屿川半开玩笑道。

但刚说完他就想扇自己一个大鼻窦。

毒舌的老毛病又犯了....

“都这样了你还欺负我...”，林穗安眼睛垂了下来，破罐子破摔道：“行，秧秧书包里装的是内衣和胖次，苏苏酱满意了吧？”

“...”，苏屿川自杀的心都有了。

毕竟是出门旅行，稍加思索便能想到包里带的是换洗的衣物，自己还非要迟钝的刨根问底，简直不要太讨人厌。

“那我帮你背吉他吧。”，苏屿川说罢便示意林穗安将琴包拿下来。

见苏苏酱真是在为自己考虑，林穗安也感到幸福。

想怎么欺负都随他吧，只要他还喜欢自己，秧秧受着就是了。

“谢..谢谢，好き（喜欢）。”，林穗安小声嘟囔道。

“滑板？（ski）”，苏屿川满脸疑惑。

“笨蛋...”，林穗安轻轻的弹了他一个脑瓜崩。

...

庐州离洛城有很长一段距离，就算是坐动车也要将近四个小时，林穗安的吉他并不好存放到行李架上，为了避免二次破损，苏屿川只能将琴包护在身边。

动车上的空调隔绝了车厢外的高温，两人躁动的心也在上车后慢慢平静了下来。

“苏苏酱准备去玩几天？”，虽然路程遥远，但林穗安坐下后并没有急着玩手机解乏，而是向苏屿川问起了旅行规划。

她身为女孩子，出门在外要顾虑的事情还是蛮多的，爸爸妈妈那边都要知会一声。

“三天吧..七月八号回去。”，苏屿川思索后做出了回应。

“好的。”，林穗安乖巧的点了点头，旋即像是想到了什么，坏坏的打趣道：“话说...苏苏酱今天怎么不打瞌睡了？”

“哈？”，苏屿川有些不明所以。

“前几次坐火车你还蛮爱占我便宜的，这次变老实了秧秧还真有些不习惯呢...”

闻言，苏屿川老脸一红，装傻充愣道：“那是苏屿川做的，跟苏苏酱没有关系。”

“哦。”，林穗安对这个回答并不满意，随后便不由分说的把头靠到了苏屿川的肩膀上，“那这次换我来。”

“...”，感受着肩膀上平添的重量，苏屿川的神经系统迅速麻痹。

虽然这样的经历并不是没有过，但只要对象是林穗安，哪怕是最微不足道的肢体接触他也会意乱情迷。

“秧秧...女孩子要矜持。”，苏屿川摸了摸她的脑袋。

“这是林穗安做的，和秧秧没有关系。”，林穗安有样学样。

“好吧，但我的肩膀可没有秧秧的肩膀靠着舒服，你也不嫌硌得慌。”，苏屿川无奈道。

“那咋啦？”，林穗安贴得更紧了些，“我喜欢。”

她就这样不知廉耻的靠了一路，一开始精神抖擞的玩了会儿手机，这之后百无聊赖的看了会儿小说，最后还真的昏昏沉沉睡过去了。

听到林穗安的鼻息逐渐平稳，苏屿川这才敢探出脑袋直勾勾的欣赏她。

秧秧的脸上既不长痘也不长斑，唯一与白皙的肌肤形成反差的只有眼角的一颗小痣，但长的位置很不错，完全可以算是加分项。

睫毛好看，眼尾好看，鼻子好看，嘴巴好看...哪哪都好看。

太鲜美了，如果真被鬼迷心窍的话，苏屿川估计会忍不住上去偷亲一口。

但爱人如养花，苏屿川对待林穗安更是疼爱有加，最后也只敢用手指轻轻捏了捏她下巴上的软肉。

嫩得没边，又带点婴儿肥，可爱又勾人。

...

“秧秧现在应该继续装睡吧？果然只能继续装睡吧？”，此时此刻，看似睡着的林穗安的心理活动却是十分热闹。

她可没有苏屿川那么大的心脏——靠在喜欢的人的身上怎么可能睡得着嘛！？

苏苏酱在干嘛？！他是捏了自己的下巴吗？

...

“好痒啊..”，林穗安下意识就要睁眼，但还是拼命的克制住了。

那样一定会吓到苏苏酱的吧？...万一他对偷偷摸摸占秧秧便宜这件事有阴影了，秧秧以后的幸福会大打折扣吧？

自己果然只有装睡这一条路了嘛？

到底为什么要玩心大发的试起装睡play啊？！这不是活生生把自己往火坑里推，往鸿门宴上送吗！

人为刀俎我为鱼肉也不过如此吧？！

秧秧好可怜..秧秧又被欺负了..
