**第76章 你抱得太紧了..我喘不过气..**

“大家再稍微等一等，复习一下谱子，我去搬几张椅子，一会儿就可以排练了。”

午后，见众人皆是一副跃跃欲试的模样，苏屿川也不矫情，说罢便开始在储物室布置起来。

闻言，所有人都捏紧了拳头。

比起紧张，更多的还是期待。

毕竟他们这个儿戏似的乐队能走到今天已经很不容易了，足足花了七十多章的时间呢！

“队长，外设还够吗？”，见苏屿川在认真的调试设备，沈音符探过头问道。

“托你的福，够的不能再够了。”

小富婆沈音符买完键盘后，乐器店老板赠送的设备不可谓不是为乐队锦上添花，江露白的贝斯都因此配上了音响，只可惜还没人来弹。

“这次练习的曲目大家还记得吗？我前几天通知过的。”，待众人都站上了自己的位置，苏屿川最后询问了一遍。

“嗯。”

“明白的。”

“包的呀队长！”

听到三人肯定的回答，苏屿川闭上眼，深吸了一口气，一旁旁观的余观棋三人也不自觉的屏住了呼吸。

“三”

“二”

“一”

苏屿川的声音不卑不亢，每一句都敲在众人心上。

话落，房间内的噪音在刹那间尽数消失，空气中，只剩下乐器线与音响连接而产生的滋滋电流声。

...

电吉他率先发难，打破了寂静的氛围，最先开始的，是节奏的铺垫，为即将出场的人声部分打下基础。

“北风毫不留情”

“把叶子吹落”

“脆弱的她选择了逃脱”

随着林穗安的歌声响起，孤独的弦音终于有了它的归宿，一切的等待都开始明了。

三句唱完，电吉他的过载音色渐收，一鲸落，万物生，沈音符的键盘开始融进了歌声。

琴键的每一次按动都让人的灵魂与之共鸣。

....

“整个冬天 北风的痛没人能说”

有了人声和键盘，苏屿川不再是孤军奋战，往后的每一句都有浑厚的钢琴音伴随着，整个声部都和谐了不少，情感的渲染更是上了一个台阶。

...

“我从来没想过我会这样做”

...

随着副歌的到来，妄伽棠的鼓点也开始了争辉，先是鼓，再是锣，速度很缓，但每一次的敲击都充满了力量。

....

“我从哪里起飞”

“从哪里降落”

“多少不能原谅的错”

“却不能重来过”

...

「叮叮」的锣声和合成器的旋律始终伴随着林穗安的歌声，比起清唱，她此刻能明显感受到自己更容易找到最佳音准了。

这就是声部的魅力啊！

“翻开回忆角落 完美的生活”

“以为幸福都可以掌握”

鼓声响起后便没有再停的道理，到了歌曲后半段，苏屿川扭紧了音量旋钮，各种推揉弦技巧也展示了出来，感受到音调变化的林穗安也不再扭捏，大大方方的放声歌唱。

...

“原来是我（原来是我）”

沈音符的和声让人眼前一亮，虽没有林穗安那般惊才艳艳，但也使歌曲的架构丰富了不止一点。

...

“犯下从没承认的错”

仿佛要发泄情绪一般，妄伽棠发狠地甩下了鼓棒，双手在鼓与锣之间不停的变换，最为原始的敲击声令在场的所有人都受到了感染。

...

“我从来没想过 我会这样做”

“从来没爱过 所以爱错”

「all in」是指负责各声部的乐器悉数奏起，它的到来往往辉映着情绪最为激昂的副歌，而这时，恰是all in的部分。

电吉他不断地嘶吼着，苏屿川一时间竟忘记了曲谱，只剩下了手部的肌肉记忆，妄伽棠的鼓点也愈发狂躁，节奏在不经意间加快。

这些突发的变故可让负责旋律和谐的沈音符慌了神，优雅的钢琴声也随之紊乱，连锁效应下，林穗安的歌声也开始跟不上拍。

众人都听出了问题，但箭在弦上不得不发，没人敢主动打断。

很快，妄伽棠就意识到了不对，鼓手在乐队中可是同贝斯手一起起着节拍器的作用的，而当下没有贝斯手，他必须要做出改变。

他将节拍放缓，给了另外三人喘息的余地，沈音符如释重负，一点不敢耽搁，紧紧的跟上旋律，苏屿川却有些犯浑，一时间不知道该从哪弹奏。

最关键的主音吉他乱了分寸，整首歌都开始变得不再和谐。

而在这关键时刻，林穗安站了出来，她默默的抱起了木吉他，构思了几个和弦后便开始带起节拍。

懵懂的状态下听见那融洽的吉他声，苏屿川晦暗的眼神瞬间发亮，但这会儿也顾不得欣赏，迅速的跟上了林穗安的节奏，这会儿，整首歌才算是回到了正轨。

“我从哪里起飞”

“从哪里降落”

...

“（我从来没想过）”

“（我会这样做）”

“我从来没爱过（所以爱错）”

...

“多少不能原谅的错（请你原谅我的爱错）”

面对时而加快又时而急转直下的节奏，众人或多或少还是不适应，拍子是乱了又合，合了又乱，一直磕磕绊绊，就这样持续到结束。

随着沈音符松开了最后的琴键，这次的排练也落下了帷幕。

..

房间霎时恢复了寂静，若非刚才演奏的四人此刻正大口大口喘着粗气，先前的一切就好像都没发生过。

“啪啪啪！”

作为气氛组，江露白率先鼓起了掌，心情十分的雀跃，“这哪像是第一次排练啊！我靠，你们都是天才吗？！”

“哇塞！安安你太帅了！你知道吗？你拿起吉他的时候我的心脏都在跟着砰砰跳呢！！”，许昕念也猛地上前，用力地抱紧了林穗安，语气激动的没边。

“对呀对呀！小林太厉害了！”，沈音符也离开了座位，满脸都是眷恋：“吉他声一响节奏就来了，简直就是救世主的级别！”

要知道林穗安的手可是有隐疾的，拿起吉他的那一刻，可知她付出了多大的勇气。

“昕念，你抱得太紧了...我有点...喘不过气。”，林穗安脸微微发红，支支吾吾的，说起话来十分艰难。

但另一边的气氛就很差了。

...

余观棋随同演奏的两名男生一同保持着沉默。

他们远没有女生那么感性，此刻更为关注的还是演奏中出现的问题。

太拙劣了。

和他们一开始的想象根本没得比，就连余观棋一个外行都能直观的感受到，这已经不能用问题来形容了，简直就是事故。

“对不起，副歌时候我节奏完全乱了。”，苏屿川低着头，十分的自责。

他很清楚，他连训练时三分之一的实力都没有发挥出来。

“不，是我的锅，敲鼓时过于情绪化了，没能帮你们打好节拍。”，妄伽棠也紧跟着揽下了责任。

二女的发挥可以说是无可挑剔的，沈音符的节奏咬的很紧，还有林穗安那段惊艳的救场，没有她们的话，演奏怕是会更加糟糕。

相较之下，他们简直就像是来捣乱的。

“干啥呀？苏屿川你咋还臭着张脸啊？要知道你刚才已经刷新我对你最帅印象的记录了啊！”，江露白笑着拍了拍苏屿川的肩膀，用独属于她的方式安慰道。

许昕念也难得正经了起来：“小江说得对！姓苏的，你刚才特别有人样，还有小妄，真没想到你人长得文文静静的，打起鼓来狠的没边呢！”

“班长别谦虚！你就是超人好嘛！？”，沈音符目不转睛的看着妄伽棠。

“苏苏酱，赛高！”

林穗安递给了苏屿川一个微笑，表达着她的认可。

虽然她很清楚的听出了苏屿川弹奏时的生疏，但少年的这份心气她格外珍惜。

更何况，《爱错》是苏屿川在邀请自己加入乐队时二人在操场上听的歌，对她的意义不言而喻。

他一直都在重视着自己呀。
