**第79章 旮旯给木乐队？绝对达咩！**

很快，余观棋被骗上贼船的消息就在众人中传遍了，大家也都纷纷表示欢迎。

真是场及时雨。

当然，次日一早，苏屿川就把贝斯带去了学校，交付到余观棋手里。

....

新的一周

“电磁感应题你就左右手轮着来就行，找准电流方向一切都好说，再不济，我可以介绍我同桌给你认识，她物理强得可怕。”

眼看乐队蒸蒸日上，苏屿川上学的情绪都高涨了不少，教理化题目时，陈希柚光从语气都能读出他的开心。

“学长，你中彩票了吗？”，陈希柚眼睛眨啊眨，好奇的不得了。

苏屿川这才想起自己忘记做表情管理了，赶忙用手揉了揉脸让自己镇静下来，可那抹淫笑却怎么也遮不住。

“我想到开心的事了。”，苏屿川咳嗽几声后说道。

“你老婆生孩子了？”

“你老婆才生孩子了呢！”

...

苏屿川和陈希柚已经熟络到了能随意开玩笑的地步，由于清楚小妮子的为人，他也不打算瞒着。

“实话告诉你，学长把乐队组起来了，厉害吧？”

“乐队？那不是从事违法活动的吗？”，陈希柚面露难色，满脸都写着质疑。

苏屿川无奈极了：“你的偏见也太大了吧？！”

虽然在上个世纪，摇滚乐队差不多就和「非主流」与「叛逆」挂钩，但他的队员可都是新时代好青年啊！

“嘿嘿，我就随口一说，学长的乐队叫啥名啊？”，陈希柚挠了挠头转念问道。

这个问题可让苏屿川犯起了难——他们还真没想过乐队的名字呢！

真是百密一疏。

“从长计议...”，苏屿川尴尬道。

陈希柚一脸恍然大悟的表情：“从长计议乐队嘛？好奇怪的名字。”

“。”

苏屿川再也不想和这个神经大条的学妹拉呱了，心不在焉的教完了最后的几道题，也开始思考起了队名一事。

总而言之，还是得先和大家商量吧。

...

“对啊！我早就想说了！视频号的名字一直取不了，姓苏的你个队长怎么当的？！不行换我来！”，许昕念当即炸了锅，怒斥起了苏屿川。

“想说不说等于没说，你这个经纪人就称职了？”，苏屿川反呛了她一口。

“老娘不管！你现在就给我想！我还要发视频呢！”，许昕念越说越蛮横。

后桌的江露白见二人一副剑拔弩张之势，赶忙的打起了圆场：“桥豆麻袋！这高低算是个天大的事，不可急于一时，大家这周慢慢想吧，双休的时候再交换意见。”

“同意。”，余观棋点了点头。

“嗯，队名很重要的，要不然我们各自想一个吧，最后统一一下意见。”，林穗安也同意了江露白的提议。

一个好的队名应满足两个条件：能给观众留下深刻印象，能展示乐队的曲目风格。

“知道了，我待会也跟sugar和沈同学知会一声。”

...

班会课

“下周一就是元旦节了，和这周末一起连休三天，孩子们！跨年了！你们离高考只有不到六个月了！回家给我抱着书啃！”

班主任还在讲台上慷慨陈词，苏屿川的心思却飘了不知多远。

他根本专心不下来，满脑子都是队名的事。

真可谓无所不用其极。

就连「旮旯给木乐队」这么荒唐的名字他都有考虑过，但就是找不到一个合适的。

更多的时候，他尝试将队员的名字拼凑起来，还因此闹出了不少笑话。

比如江露白和余观棋，各取中间一个字，就是「露观」组合。（狗头）

再比如许昕念和林穗安，各取中间一个字，就是「昕穗」（心碎）组合。

还有妄伽棠和沈音符，稍加变通就是「棠音」（躺赢）组合。

他还是第一次觉得语文这么有意思。

哎嘿。

话说，不知不觉都要元旦节了啊。

离寒假也不远了，下周回来应该也快要期末考试了吧？

。

..

周六，众人如约到了苏屿川家进行排练。

在林穗安的建议下，他们并没有好高骛远，而是将上周排练的《爱错》又回顾了一遍。

由于余观棋刚接触贝斯不久，复杂的编曲不好驾驭，恰好，《爱错》这首歌的贝斯音不算突出，林穗安临时教了几个根音便上阵了。

“贝斯最重要的还是稳住节奏，虽然这次只是拨这几根弦，但一定要配合小妄的鼓点，大家的节奏就全靠你们了！”，林穗安摆出了一副认真脸。

苏屿川默默的欣赏着她的说教，他知道，这只是林穗安的伪装，背地里，这个“现充班长”就像是变了个人。

谁又能把这个不苟言笑的班长和元气满满的「秧秧」挂钩呢？

“还有你！苏苏酱，这次可给我认真一点！实在是听不懂鼓点就跟着我的伴奏吉他走。”，林穗安见苏屿川一副事不关己的模样也教训起了他。

这会儿，众人听到林穗安称呼苏屿川为“苏苏酱”已经见怪不怪了。

“哇嘎哒。”，苏屿川点了点头。

...

（乒乒乓乓咚咚锵锵）

一回生二回熟，这次的排练效果出奇的好，一旁负责录像的许昕念一脸的满意。

旁观的江露白也很是激动的鼓起了掌——余小样认真的时候真的很帅啊。

....

“余同学真的是第一次学乐器吗？节奏跟的十分好呢！”，复盘时，林穗安有些惊喜道。

“谢谢。”，余观棋回答的不卑不亢。

“小妄的鼓点和小沈的键盘也都很稳呢。”

“谢..谢谢。”

“嘿嘿。”

轮到苏屿川时，她的态度突然急转直下。

“苏苏酱，你又得意忘形！”，林穗安不禁皱起了额头，“副歌空白期你的吉他声音怎么比鼓都大，就这么想solo？”

“略略略。”，苏屿川不买账。

对于林穗安的偏心对待，他早就见怪不怪了，毕竟每周都要去她家开小灶，自己几斤几两她比谁都清楚，这种程度的教训只能说是习以为常。

打是亲骂是爱。

...

休息片刻后，苏屿川开始征询起了大家想的乐队名。

“翡翠椒烤鱼乐队！”，江露白率先表决。

“饿了自己去吃，否了。”

“魔法少女乐队！”，许昕念紧随其后。

“乐队还是有男生的，否了。”

...

“糖音炮弹乐队！”

“挺好的...可惜没啥记忆点。”

...

“慈禧的秘密乐队。”

“余观棋你有病啊？！”

...

最后，苏屿川只能把希望寄托在两大班长身上了。

妄伽棠和林穗安还没发话，想必才华横溢的二人要更权威些。

“die $\pi$乐队怎么样？我看最近在网上挺火的。”，妄伽棠一脸期待道。

“...”

“。”

“。”

“。”

“。”

“？”

六脸懵逼了属于是。

“sugar...你知其火不知其所以火啊，我敢打赌，乐队取这个名我们走到哪都是别人的笑柄...”

“啊哈？”，妄伽棠满脸的震惊。

“我..我想的是英文名。”，林穗安见自己压轴出场，不禁有些紧张，“但我看网上都是这样的。”

说是在互联网上学的，实际上林穗安都是参考的galgame，不过，galgame里的，应该也没啥问题吧？

“Finally·LoveAndDeath·Forever。”

“你想的是裹脚布吗？这怎么听都是个非主流乐队吧？网上真的有这个？你刷的是暗网吗？”，苏屿川最后的希望也破灭了。

“姓苏的，你别就指着鼻子喷我们啊，你自己想的呢？”，许昕念见他不识好歹，竟然连安安也喷，当即就怼了起来。

“呃..哈哈，呃..”，苏屿川顿时犯起了难，他还真没啥好点子。

总不能叫“躺赢，心碎鹿关”乐队吧？

但见六人都目光灼灼的看着自己，他怎么着也得说点什么。

“旮..旮旯”

“绝对达咩！”，林穗安脸霎时就红了，别人猜不到下文她还能猜不到嘛？

就这样，这次取名行动无疾而终了。
