**第31章 因为，你是特别的。**

“击弦....这个练得差不多了，再然后是泛音，分为人工泛音和自然泛音。”

苏屿川独处时有自言自语的习惯，体育课上，当同学们在球场上尽情玩耍时，他却是一个人躲在不起眼的角落翻看吉他书。

换作以前，他还能和余观棋，江露白二人吹吹牛，但最近那两人就跟打了兴奋剂似的，两眼一睁就是学习，体育课都不来上了。

比起花里胡哨的加花技巧，苏屿川知道，他的当务之急是学会看谱子，于是乎，他就抱着一本《电吉他入门经典教程》在那啃。

这本书收录了不少知名歌曲的吉他谱，后面又配有解析，虽然对初学者来说有些晦涩难懂，但肯下功夫效果还是非常显著的。

“先是四弦九品的无头滑音，然后三弦九品到十一品的击弦.....”

苏屿川嘟囔着，右手翻书，左右虚空滑动，似乎是在对应吉他上的品格。

“嗯，然后是弹三弦十一品，再配合九品到十一品的击弦，最后弹二弦十品。”

后面的谱子也被破译了出来，只不过说这话的人不是苏屿川。

不知何时，林穗安站到了他的身后。

“？！”，苏屿川的惊诧都写在脸上，吓得一股脑把书丢到了地上。

“干嘛？只许州官放火，不许百姓点灯啊？不就是看你一个谱嘛，你好大的脸啊。”，林穗安也被苏屿川过激的举动吓到了，悻悻的将书捡起来后不禁轻声埋怨道。

“你啥时候来的？”

苏屿川的心怦怦直跳，自己原本的想法是悄悄学会电吉他，然后惊艳林穗安，没想到这会儿被当事人抓了现行，他又怎能不被吓到？

“从你鬼鬼祟祟的在这边坐下开始。”

“...”

那不就是一直都在吗？

完了，一切都完了。

计划彻底破碎了。

“伤口涂了吗？”，林穗安没搭理他那生无可恋的眼神，说罢，目光看向了他的左手。

“啥伤口？涂啥？”，苏屿川前一瞬间恍如隔世，此刻有些摸不着头脑。

林穗安见他那傻样又好气又好笑，伸出了自己的左手说道：“我碘伏又不是给你吃的，你食指再不处理的话，水泡破了，想练琴都练不了。”

“练啥琴？”

苏屿川这才知道林穗安给自己碘伏是处理伤口的，他原以为是她乱丢的呢，还准备等她气消了再还给她。

“你别跟我装傻，早上翻你书包的时候看到这本书了，你不是在练琴，还能是在学习敲摩斯密码吗？”，林穗安不禁白眼。

苏屿川这才知道他彻底露馅了，一时间不知该如何是好。

他有种强烈的受挫感，这种感觉就像是，你精心去准备一件礼物，想要给某人一个惊喜，结果在企划案刚写下来的时候，发现这个消息被泄露了。

他是因为林穗安才练琴的，结果却在胚胎时期被她一窝端了，心里憋着的那股一鸣惊人的劲头瞬间就少了一半。

“呵呵，实则不然恰恰相反，我马上回去就把吉他给卖了.....”，苏屿川像是失了魂，两眼空洞的说完这句话后就准备逃离。

一定是幻觉...眼前的林穗安一定是幻觉，百密一疏不可能发生在我身上，假的，都是假的，她什么都不知道，我还能接着为她准备惊喜。

对，练琴太投入练出幻觉了。

“不准卖。”

苏屿川刚起身，林穗安的小手就拽住了他的衣角。

“虽然不清楚你为什么学吉他，但是刚学不久就放弃，以后一定会后悔的...就像我一样。”

林穗安说这话时，语气再没有过往的俏皮，表情也是格外的真挚。

“哦，那不卖了。”

苏屿川认命了，说完又在林穗安的身边坐了下来。

“戴耳机了吗？我想听音乐。”

林穗安见他有意回避有关「吉他」的话题，索性便没再提及。

“喏，给你。”，苏屿川也不磨叽，连上手表后便递过一只耳机给她。

“我要右耳的。”，林穗安没有接。

“有区别吗？”，苏屿川不解道。

林穗安微笑着歪了歪小脑袋，“有啊，你坐在左边，我戴右耳你才能听清我说话呀。”

“...”，林穗安不经意的一句话让苏屿川倍感害羞，别扭的把头转了过去。

真是个对心脏不好的家伙。

耳机里，是王力宏的《爱错》

「北风毫不留情」

「把叶子吹落」

「脆弱的她选择了逃脱」

....

这一回，林穗安首次在听歌时和自己聊起了天。

“苏苏酱，你喜欢音乐嘛？”

“嗯。”苏屿川回答的很笃定。

由于一向寡居，他养成了听音乐和自言自语的习惯——在不同的歌声中寻求慰藉，在多样的节奏中发现真我，似乎只有这样，才能或多或少的缓解他的孤独。

其它的他不敢保证，但喜欢音乐这一点他对自己有着十足的信心。否则他也不会以“乐队”为切入点，尝试吸引林穗安。

...

「我从来没想过 我会这样做」

「从来没爱过 所以爱错」

「我从哪里起飞」

「从哪里降落」

「多少不能原谅的错」

「却不能重来过」

....

耳机里的音乐还在继续，一如林穗安的话语没有终止。

见苏屿川的芥蒂少了些，她开始分享起了她的故事。

“其实，我初中学过三年的弹唱，我没有告诉任何人，就想着在毕业典礼上装一波大的。”

回忆起过去时，林穗安的眼神里满是眷恋，像是在怀念那个小小的自己。

“那会儿我除了学习，剩下的时间都在练琴，指尖磨破了一次又一次，洗手都刺辣辣的疼，刚开始那一阵子，我每次都是疼的一边哭一边练，现在想想也真够傻的。”

林穗安说到这，浅浅的笑了笑，富有青春气息的梨涡显得格外可爱诱人。

「多少不能原谅的错」

「请你原谅我的爱错」

...

“后来呢？表演了吗？”，音乐终了，苏屿川看着林穗安的侧脸，缓缓开口问道。

林穗安苦涩的摇了摇头，“后面发生了一些事，左手动了手术，之后，食指和中指活动就不灵活了，连弹最基本的和弦都费劲。”

气氛一时间陷入了沉默。

...

“那...你后悔吗？”，思虑良久，苏屿川开口问道。

他没想到林穗安这么乐观的女孩还有这么段悲伤的往事，内心感触良多。

林穗安轻轻摇了摇头，将脑袋偏了偏，似乎是不想让苏屿川看见她眼中的泪珠闪烁。

“哪方面的后悔呢...后悔学吉他吗？我从来没后悔过哦。”

“不过，一直犹犹豫豫，没有及时记录倒是蛮遗憾的，早知道就不那么死脑筋的追求结果了，其实现在想想，过程中的每一次进步，学会的每一首曲子，每一个动作形成的肌肉记忆都值得记录。”

不盲目的追求结果，而是享受追逐的过程...

林穗安的话点醒了苏屿川。

他想通了，他完全没必要介怀，为何要执着于“一鸣惊人”呢？能在喜欢的女孩子的见证下进步，又何尝不是一种殊荣？

“所以你才会这么憧憬乐队吗？”，苏屿川将林穗安的过去和现在联系起来，才发现一切似乎都有迹可循。

林穗安点了点头，“是的，我好羡慕她们啊，享受着美好的青春，拥抱着诚挚的友情，追逐着闪耀的梦想，但我不敢说出来，只敢在别人看不见的地方偷偷喜欢。”

“那我算是你的变数咯？”，苏屿川回想起自己和林穗安的初次邂逅，忍不住发问道。

自从在阴差阳错下发现林穗安的小秘密后，二人之间的羁绊就越来越深了。

“是又不是。”，林穗安给了模棱两可的答案，“一开始我觉得你可讨厌了，就爱欺负我，拿我的秘密说事，但现在我有所改观了。”

“怎么说？”，苏屿川好奇道。

“因为，你是特别的，或许那晚在书店的相遇是冥冥中的注定呢？你和我一样喜欢音乐，我们又偶尔都幼稚得烦人，甚至你也和从前的我一样喜欢上了吉他，这些巧合都让我觉得，你是特别的。”

林穗安说这话时目光始终没有偏移，这些煽情的话并不是少女的娇羞告白，而是得遇知己的肺腑之言。

苏屿川在林穗安的眼神中看不到恋人的情愫，自然也不会把这当做告白，她愿意和自己说这些，只是把他当做知心朋友而已。

“我学吉他的目的没有你那么高尚，如果仅凭一腔热血的话，我想我早就放弃了。”，苏屿川笑了笑，叹了口气后说道。

能让我坚持下来的，是你啊。

“苏苏，加油，带着我的份一起努力吧。”，林穗安也扭头冲他笑了笑。

直到这时，苏屿川才看见了她眼眶满载的眼泪。

...

「手心的蔷薇」

「刺伤而不自觉」

「你值得被疼爱」

「你懂我的期待」

「绚烂后枯萎」

「经过几个圆缺」

「有我（你）在」

...

耳机里响起的，是《手心的蔷薇》的副歌，苏屿川慌忙的摸了摸口袋，这一回，他总算找到了纸。

“谢谢，让你见笑了，听我倒苦水什么的。”，林穗安轻拭眼角，有些不好意思道。

“林穗安，你..还想弹吉他吗？”

在购买吉他前，苏屿川就曾有过幻想，而现在，他觉得这或许能成为现实。

“想呀，可我的手不听话。”，林穗安说罢，破罐子破摔似的伸出了左手。

她的手格外好看，兴许是多年没碰琴了，指尖已经恢复了细嫩，指关节也修长白皙，唯一突兀的，是虎口上方的一道疤痕。

“它不听话，那我们就教它听话！”，出于激动，苏屿川轻轻的握住了她的手掌。

“林穗安，我很认真的邀请你，来我的乐队做主唱吧！”
