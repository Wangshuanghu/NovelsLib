**第91章 有学长陪我真好～**

天黑前，苏屿川在车站告别了李澜夏。

这会儿只剩下陈希柚与他同行了，学长的学长走了之后，柚子学妹明显轻松了不少。

“呼，今天差点想死在那了。”，陈希柚晃着琴包，走起路来东倒西歪的，看起来笨蛋极了。

苏屿川嗤笑了一声，淡淡开口道：“那看来柚子比较适合玩摇滚乐啊，毕竟摇滚的最高荣耀就是死在舞台上嘛。”

“绝对不要！”，陈希柚吓了一跳，她只是说着玩，哪里真舍得死啊？

苏屿川见学妹呆头呆脑的也懒得再打趣了，转念问道：“明天你还要来吗？我可以过去给你捧场。”

“真的嘛？”，陈希柚本来心里挺没底的，有了苏屿川这话反倒振作了不少，“那学长可以带吉他来弹吗？两个人的话应该就不害怕了...或许。”

这倒是点醒苏屿川了。

虽然每天都练吉他练得热火朝天，但他至今还没有当众演奏过呢。

话说..由于一直是在跟着手机自学，他的团队配合能力实在是有待提高。

这个弊病在乐队排练的时候暴露无遗，抢拍，慢拍都是常有的事，常常弹着弹着他就只能听到自己的声音，导致许昕念采录的视频里，电吉他的声音尤为突兀。

这也是他被骂吉他嘉豪的根本原因。

“行啊，咱俩可以搞个街头live，不过事先说好，我的吉他功底可不深，你也看到了，短视频平台上都快被喷成筛子了，拖后腿了别怪我啊。”，苏屿川顿时来了兴趣。

他可不像陈希柚那么社恐，比起恐惧，内心的激动显然是更胜一筹的。

“不怪你不怪你！我受着还不行嘛。”，陈希柚开心极了，“这样的话要尴尬也是一起尴尬，嘿嘿，有学长陪我真好。”

...

晚上

「来抓我呀杂鱼」向你发出了组队申请。

苏屿川歪歪脑袋——他刚准备上号召唤杂鱼小姐呢，没想到她竟然不请自来了。

“屿川大王，你不乘哦，一下午都没上号，是不是不想好了？”

刚进队伍，林芷乐就开始兴师问罪。

单排打排位实在无聊，于是她就打了一下午的巅峰赛。

这会儿游走的巅峰分都到2480了。

“打一下午游戏才是不想好了吧...”，苏屿川吐槽了一句，“还有，你一下午都上了130分，就别再得了便宜还卖乖。”

苏屿川在2300分卡了好久了，一直没有进展，可杂鱼小姐一下午就从2300上到了近2500。

她的战绩都是全开放的，巅峰赛十连胜的记录尤为醒目，每把都有牌子。

虽然杂鱼小姐不怎么打巅峰赛，但她实力是毋庸置疑的。

苏屿川隐隐感觉她的水平还要在李澜夏之上。

“嘿嘿，随便拿捏好吧。”，见苏屿川被自己装到了，林芷乐难免有些得意。

虽然她的强大是事实，但苏屿川才不会认账呢。

“狗运罢了。”

“你才狗运呢！”，林芷乐来了脾气，“再说了，要不是你装死我能去打巅峰赛嘛？废话少说，开开开。”

苏屿川始终记着李澜夏的嘱托，进游戏后，在bp阶段淡淡开口问道：“那个，今年三月有时间吗？”

“干嘛？”，林芷乐轻咦一声，思索良久后补充道：“那一阵子还真没啥事。”

“那就好。”，苏屿川松了一口气。

“上次三排的那个玩西施的学长还记得不？他组了个战队，准备打线下比赛，地点在江宁，第一名队伍有五位数的奖金，看你辅助玩的好，想要邀请你加入。”

毕竟这不是像APL那样的职业联赛，凭借他们的实力，哪怕对抗和射手栓条猪来，要夺个名次也是没啥问题的。

林芷乐回忆了起来——元旦节玩西施的那个冒失鬼她哪能不记得？当时都给她逼得开麦了。

虽然玩的是脆皮法师，还爱闪现开人，但是那几局的西施意识还是很顶尖的，死亡率也很低。

很强——这是林芷乐对李澜夏的评价。

不过她可不在乎这个。

...

“那你会来吗？”

“嗯。”

“噗，你的学长咋还雇童工啊。”

林芷乐轻声笑了笑，并没有急着给他答复。

“赛事名称是啥？虽然挺想凑这个热闹，但还要再看看才行，事先说好...我可能不符合参赛标准。”，林芷乐幽怨道。

“哈？”，苏屿川百思不得其解，这只是民间的比赛，审核的要求很宽松啊，也没有性别的限制，“难不成你是未成年啊？”

“滚一边去，你又不是不知道我多大。”，杂鱼小姐骂了他一句。

她的生日在八月，只比苏屿川大十来个月，周岁十八虚岁十九。

“那不就行了，你总不能不是我国人吧？”，苏屿川回怼道。

林芷乐叹了一口气：“算了，和你说不清楚，你待会儿加我QQ，回头再详谈，先打游戏吧。”

两人一起玩了快半年了，到现在才想到要互换联系方式。

“行。”，苏屿川见她有在认真的考虑，便识趣的不再多说什么，“对了，打三把我就要下了，约了朋友玩我的世界。”

“沙盒？”，杂鱼小姐语气有些惊讶，“屿川大王，你可真是童心未泯。”

“对啊，我是未成年。”

...

「予川」向你发出了好友申请。

几局游戏结束后，苏屿川如约加了林芷乐的QQ。

【Forest】：予川？屿川大王，你的马甲可真多。

【Forest】：（竖大拇指表情包）

【予川】：...我的大名是苏屿川，也不算是马甲吧？

【Forest】：（震惊），原来真有人实名上网啊？！

...

苏屿川也臊红了脸，每次跟同学打游戏都要被拿游戏名字说事。

无敌屿川大王...

【予川】：那你呢，你叫啥，我总不能一直叫你杂鱼小姐吧？

另一边

林芷乐无奈的叹了口气。

按理来说，苏屿川都做完自我介绍了，二人现在应该礼尚往来的交换名字。

可她有不能透露的理由。

【Forest】：可以呀，就叫杂鱼小姐呗，反正我听习惯了。

【予川】：？？

【Forest】：...我姓林，剩下的不能再说了。

苏屿川愣了愣。

原来还和林穗安一个姓呢。

【予川】：行，那我以后就叫你林杂鱼了。

【Forest】：666，小学生，你开心就行。

【予川】：（链接）

【予川】：我找学长要来的比赛链接，如果没问题的话我们一起报名吧。

林芷乐点开后认真的阅览了一番。

1.参赛者必须年满18周岁（含），年龄判断以身份证出生日期为准。

2.以队伍为单位进行报名，至少五人一队，至多可以报7人队伍，队伍成员性别不限，可自由组队。

3.报名时务必填写真实、准确的个人信息，不允许出现顶替上场的行为。

....

“呼，吓死我了。”，林芷乐仔仔细细的从头看到尾，确认这不是官方赛事后才放下心。

不禁止现役职业选手参赛就行。

【Forest】：看完了，我能上。（得意）

【予川】：这不是废话吗，只要是个人都能上吧？

【Forest】：那可不一定。

苏屿川自然不会朝林芷乐是职业选手这个方向去想。

即使她的实力拔群，但终归也只是个十九岁的女生啊！

虽然苏屿川并没有详细了解过王者的赛事，但他知道电竞的赛场上差不多都是男性。

【Forest】：对了，队里还差人吗？我可以给你搬救兵哦。

见状，苏屿川惊喜的不得了。

这可真是雪中送炭，困了有人递枕头啊！能被杂鱼小姐称作“救兵”的人，那该会强成什么样啊？！

【予川】：差！还缺射手和对抗！

看完，屏幕对面的林芷乐嘴角不自觉的上扬。

射手和对抗啊...那事情可要变得有意思了。

和她同战队的队友可是被誉为对抗路最高的山和发育路最长的河呢。

【Forest】：了解～屿川大王，等着到时候领奖金吧。

【Forest】：（魔丸表情包）

【予川】：嘿嘿，包的。
