**第108章 小小的也很可爱哦！**

“唉。”

回到家后，江露白止不住的叹气。

她也不知道自己这是怎么了，明明不想和余小样闹成这样，但自尊心总是在作祟，让她无论如何都不能再选择低头。

余观棋这次太过分了！！

虽然袭胸什么也是始料未及...但“锉刀”是个什么鬼啊？！

想到这，她还不服气的用手揉了揉。

好平，青青草原似的。

...

“我平平无奇又不是一天两天的，干嘛非要拿这个说事啊...”，江露白越想越郁闷，“我也是有其他长处的好嘛？”

就比如...

呃。

想不出来。

“好嘛，我就是小矮个、飞机场、脾气臭、男人婆、差等生，行了吧！”，江露白都快被自己气笑了。

越想越是心烦，最后她选择了打开恋综。

窥探别人谈恋爱是她为数不多的乐趣。

看嘛！哪有那么多女生天生丽质的？！飞机场也是很有市场的好嘛？小小的也很可爱啊！

小小的..也很可爱啊。

。。

虽然自己的尺寸可能...可能确实低于平均水平，但也是有在长大的好吧？！

平胸咋啦？平胸吃你家大米啦？

...

“唉。”，江露白叹的气就没停过。

房间外，江晖正烧着午饭，朱青玲则像女王似的窝在沙发上看电视剧，两人皆是没有注意女儿的异样。

虽然江露白一有委屈就会主动跟父母报告，但“锉刀”一事她实在是开不了口。

也幸好江露白没敢说，不然女儿奴江晖肯定要把余观棋砍成臊子。

“白露，吃午饭啦。”，过不久，妈妈朱青玲语气温柔道。

虽然她对江晖的态度很是刻薄，但对女儿却是打心眼里疼爱的，从小到大一点苦都没让江露白吃过。

“知道了妈妈。”，江露白做了个深呼吸，尽量让自己的语气保持正常。

....

“白露多吃点，高三最后一学期了，好好加油啊，等高考完了爸爸带你出国旅游。”

餐桌上，江晖像是个男仆，一会儿给老婆夹菜一会儿给女儿夹菜。

“呜，daddy，我觉得我撑不到高考了。”，江露白苦了苦嘴巴，“我好累啊。”

闻言，朱青玲和江晖皆是吓了一跳。

白露一向活泼得很，虽然也经常抱怨学校抱怨老师，但如此消极还真是头一回，更何况今天还是开学第一天。

“咋啦宝贝？”，朱青玲有些担忧。

江露白也不知道该如何说明，如果欺负她的是别人，她肯定没啥顾虑，但余观棋的身份太犯规了。

“nothing，nothing，我随口说说而已。”，江露白挤出一抹笑意，旋即装作没事的吃起了饭。

江晖和朱青玲对视了一眼，双双察觉到了不对。

女儿闷闷不乐已经不是一天两天了，从过年一直持续到现在，肯定是有心事啊。

但江露白不说，身为父母的他们自然也不方便去问。

饭后，趁江露白午睡，夫妻俩开始悄悄讨论了起来。

“丫头最近很不对劲啊。”，朱青玲皱着眉头道。

江晖点了点头。

见状，朱青玲气不打一处来，一拳就塞到了他肩膀上：“你别光点头行不行？能不能想想办法啊？把我那个活泼开朗的女儿还给我！”

“我能有啥办法呀，我又不是肚子里的蛔虫，孩子在想什么我上哪知道啊？”，江晖委屈极了。

余向前，苏承荣和他三个大男人，就属他的家庭地位最低。

“你在这跟我甩上脸子了？”，朱青玲本就不爽，见丈夫还敢回嘴更是火冒三丈。

江晖嗅到了一丝杀机，心有余悸的回答道：“没有，请长官明示.. ”

“这还差不多。”，朱青玲这才收起了气场，“丫头跟观棋和屿川的关系不是挺好的嘛？你就让他俩平时多担待一点呗，孩子们之间的共同话题肯定也多些。”

“好主意啊。”

“那当然，我可是天底下最聪明的女人。”

....

既然老婆大人都发话了，江晖也不敢耽搁，当即便敲响了余向前家的门。

邻居就是方便。

“老余，观棋在家吗？”

余向前开门后，江晖直奔主题道。

“在..倒是在。”，余向前有些诧异，一般江晖来拜访都是找他喝酒，为了找余观棋还是头一回。

“孩子睡了吗？”，江晖见时候也不早了，担心余观棋还在午睡。

余向前摇了摇头，他们家饭吃的晚，这会才刚收拾好。

“咋啦？”

“没啥，孩子间的事，我去找观棋扯几句。”，江晖冲余向前嘿嘿一笑，旋即便走到了余观棋的卧室门前。

虽然门没关，但江晖还是敲了敲，给余观棋留以反应的余地。

“江叔？”，余观棋摘下耳机，见到来人后满脸都写着吃惊。

不会是来找自己兴师问罪的吧？

“观棋啊，叔跟你说点知心话。”，江晖对余观棋这个娃子还是很满意的，尤其是上次看了他的贝斯演奏。

余观棋忍不住咽了口唾沫。

这是要碎尸万段的前奏啊！

“你是白露的同桌吧？最近她是遇到什么事了吗？感觉最近不是很开心。”，江晖说话好声好气的，没有一点居高临下的意思，仅仅只是站在一位父亲的立场。

余观棋不禁愧疚了起来，江叔叔不知什么原因他还能不知道吗？

只不过这话太难以启齿了。

“我...”，余观棋一时语塞。

“观棋，算叔叔拜托你，你平时多担待担待，她现在大了，有些话我们也不方便说...我家女儿把你看得挺重的，你知道吧？”

江晖也不知此话当讲不当讲，但江露白喜欢余观棋这一点，在他们的眼里还是很明显的。

就比如江露白在和他们分享学校日常的时候，三句中有两句都不离余观棋。

但总不能提倡孩子早恋吧？

余观棋本就被苏屿川的话说得晕乎乎的，江叔叔这么一补充更是让他如遭重击。

“我，我知道了。”，余观棋木讷的应道。

江晖也是明事理的人，孩子的事终究是孩子的事，他也不可能玩道德绑架那一出，接下来二人的对话主要还是围绕贝斯展开。

为了不被父母发现，余观棋一直将贝斯藏在床底，平时也只敢在半夜练。

但好在贝斯的声音不是很高，房间的隔音效果也不错。

“真好啊，有我当年的风范，哈哈哈。”，江晖越看越满意。

余观棋做事很有恒心，天赋也不错，在一些专业术语的交流中甚至能做到有来有回。

面对江晖的夸赞，余观棋始终有些专心不起来。

他越发感到内疚。

“江叔，那个，能不能帮我转告一些话给江露白。”
