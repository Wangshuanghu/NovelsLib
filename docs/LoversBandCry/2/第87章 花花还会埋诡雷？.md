**第87章 花花还会埋诡雷？**

“喵呜～”

花花被放在怀里捂了一路，临到家时，苏屿川还给小家伙买了杯纯羊奶。

“你哥对你真没得说，冻死了都不忘给你保暖。”，苏屿川一边打着喷嚏一边自嘲道。

妄伽棠平时的作息很规律，这会儿已经洗漱好准备休息了，这位突然造访的小客人可给他吓一激灵。

“花花，认识一下，这是你的sugar哥哥。”，苏屿川说完将花花的脑袋转到了妄伽棠面前，像是在给它做介绍。

“喵呜～”

妄伽棠来了兴趣。

“花花？它的名字？”

“嗯，捡回来的，我们的新租客。”

妄伽棠若有所思的点了点头：“不过一般来说，小说里，主角在雪夜捡到的都是黏人的校花和破碎的少女，你捡只猫有点太丢份了。”

“所以说我不是主角啦！”，苏屿川自嘲道。

“我们都不是。”，妄伽棠耸了耸肩，“现实哪有小说那么好写。”

“sugar还跟我玩谦虚呢？小说男主可不一定考得到700分。”，苏屿川白了他一眼。

闻言，妄伽棠苦涩的笑了笑：“可男主不会以捡垃圾开场啊。”

....

两人尽情聊着无厘头的天，最后一同接了热水给花花洗澡，这天就算是过去了。

花花很粘人，熄灯后一直“喵呜”个不停，也不知它是想妈妈了还是嫌无聊了，不过，第二天醒来的时候，它已经钻进了苏屿川被窝。

回来时苏屿川一直把它抱在怀里，或许，这里于它而言有着无可比拟的安全感。

“呼..”

晨光透过窗户，打在了苏屿川脸上，看着怀中酣睡的花花，这个腹黑又傲娇的男生心都要给萌化了。

把它带回来真是个不错的决定呢。

但这样的念头没有持续太久，小家伙随地大小便的坏习惯就暴露出来了。

由于身体小，又营养不良，可以说是吃多少拉多少，一上午，家都要被它尿的漂起来了，东一块西一块。

铲屎官们真不容易啊！

外面的雪已经停了，行人也渐渐的多了起来。

阔别两年下的第一场雪，来的快，去得也快，但苏屿川一点都不惋惜，因为他在昨晚已经与喜欢的女孩度过了最浪漫的时刻。

他很开心，林穗安第一个想到的散步搭子不是许昕念，而是自己。

嘿嘿，她是不是也喜欢我啊？

带着这份悸动，他今天总算是能静下心来复习。

考试第一天

“欸，还真下雪了，奇了怪了，我一离开庐州就下雪。”，许昕念坐在考场里，失神的看着窗外道。

她和苏屿川真可谓不是冤家不聚头，明明考场顺序全面打乱了，两人又鬼使神差的坐了前后桌。

“你这两天不在庐州吗？”，苏屿川正争分夺秒的看着古诗文，回答她的时候头都没抬。

“嗯，回了江宁一趟，话说...还是安安跟我说庐州下雪了呢，前晚约我出去散步，可惜我人不在。”，许昕念有些小郁闷道。

可苏屿川就有些大郁闷了，翻页的手指都顿了顿——合着林穗安一开始邀请的不是自己啊？！

“。”

果然，暗恋的一切甜蜜都来源于暗恋者的幻想。

“咦？你语文不背了嘛？”，许昕念见他失落的站起了身，有些诧异道。

“背个鸡毛啊。”，苏屿川重重的叹了一口气，“我都失恋了。”

“莫名其妙。”

郁闷归郁闷，该做的复习工作前些天已经做过了，所以语文整体写的还算差强人意。

“嘿嘿，这次的作文真浪漫，「迎着光，追着光，向光而去」，我都快被融化了。”

交卷后，许昕念趴在桌子上感叹道。

“别迎光了，迎你的二百五十分去吧。”，苏屿川波澜不惊的收拾着笔和橡皮，临走之前不忘出言嘲讽。

“嘁，等哪天你语文考不过我就老实了。”

“那我应该是要考虑复读了。”

下午的数学卷出的很简单，俗话说「难度大不大，全看数理化」，毕竟是期末考试，为了让同学们的成绩单好看些，得以回家过个好年，出卷老师也是煞费苦心。

.....

“这物理题都是啥啊，给初中生做的吗？”，考完物理后，许昕念又开始了她的凡尔赛。

“你也只有考完物理能这么说了。”

...

“苏，我的英语好像又有点死了。”

“没事的小样，自信点，就没活过。”

...

三天一转而过，笼罩数天的阴霾也在一刻之间悉数散尽。

不管考得好不好，那都是过去式了，想开点，再上一个星期学可就放寒假了。

收拾收拾，准备回家过年吧。

憋了三天，再次拿起吉他时，苏屿川有些心潮澎湃。

这会儿也不枯燥了，底下，花花一直环绕在他的腿边，这蹭蹭那蹭蹭，可稀罕人了。

妄伽棠的估分还不错，也是如释重负的坐下来准备练鼓。

可很快他就发现了不对劲。

“屿川，这个踩镲怎么踩着软绵绵的？”，说罢，妄伽棠还不信邪的加大了脚尖的力度，但奇怪的感觉却经久不衰。

“啥？”，苏屿川扭头看了过去。

“！！！！”

“sugar，别动！千万别动！”，苏屿川下巴都要惊掉了——花花竟然把大便拉到了底鼓踏板上。

“咋了？”，妄伽棠疑惑的低头看去：“....”。

没活着就是死了。

....

“总算是清理好了，真想不到花花还会埋诡雷...”，苏屿川有些尴尬的朝妄伽棠道歉道。

明面上的排泄物清理完后，没想到小家伙还在暗处藏了宝。

妄伽棠的情绪还算稳定，释然道：“看来家里多了位小警察呢，专门监督我们的，只要不练乐器就有报应。”

“要真是这样我可得给吉他藏远点了。”，苏屿川半开玩笑道。

这么一说，储藏室里一地都是音响和乐器线，再不稍加管制的话，花花怕是得大闹天宫。

以后不在家还是把储藏室门关着吧。

一段小插曲过后，苏屿川拾起拨片，对着林穗安为她打印的谱子开始了练习。

经过近三个月的努力，他现在已经能做到无障碍读谱，无头滑音，击勾弦，揉弦啥的也都已经是老朋友了。

《权御天下》——这是他现阶段正在练习的曲子。

开篇音色便是强效失真，尤其炸裂，节奏也是快到不行，用来锻炼手指机能再合适不过了。

“东汉末狼烟不休”

“常侍乱朝野陷阿瞒挟天子令诸侯”

...

这也是StarryStar准备在寒假排练的曲目，浓烈的国风使得它在短视频平台尤为火爆，机灵的许昕念自然不会放过这泼天的流量。

歌词节奏这么快，不知道林穗安唱起来会是什么样呢？

苏屿川不禁开始期待起来。
