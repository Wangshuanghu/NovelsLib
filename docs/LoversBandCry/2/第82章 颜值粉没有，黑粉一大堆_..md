**第82章 颜值粉没有，黑粉一大堆...**

过完幸福的元旦假期，再回学校简直是要了苏屿川的老命。

一月的温度已经很低，离开温暖的被窝真的很需要决心，他真羡慕妄伽棠能那么云淡风轻的起床。

“sugar，我要死了，你能不能帮我请假啊？”，苏屿川一副死猪样，歪在床上说着胡话。

他这些天时而练琴时而打游戏的，作息十分紊乱。

“不行的，快起来吧，我刚煮好面条，一会儿凉了。”，妄伽棠有些无奈的笑道。

两人同居已经有不少天了，羁绊也愈发的深厚。

...

临近小寒，路旁的树杈已经十分萧索，稍有一阵寒风就能发出呜咽，将未尽的困意也给冻醒。

“陈老师，作业搬来了。”

早读课下，苏屿川照例把数学作业送去了陈郁办公室。

“辛苦你了课代表同志...”，陈郁回答的有气无力的，看她的脸色像是遇到了什么烦心事。

“恕我冒昧问一句...老师你这是又相亲失败了吗？”，苏屿川咧咧嘴巴道。

“小同志，成年人的世界很复杂的，哪有那么多时间相亲。”，陈郁回敬了一个白眼，“是柚子的事，她昨天在音乐班的演出上捅了大篓子。”

“啊哈？”

“明明排练的时候还表现得很不错，一到正式演出就拙态百出，已经不知道多少次了。”，陈郁说罢叹了一口气，“这样下去艺考可怎么办啊...”

闻言，苏屿川也不知该如何作答，虽然陈希柚也说过她不习惯在很多人面前演奏，但这点却是不克服不行的。

或者说，这是每一个玩音乐的人都必须经历的一道坎。

...

“所以，根据题意可以判断，这里是羟醛缩合，那么...”

大课间辅导作业时，苏屿川能明显感受到陈希柚的情绪有点低落。

这种事是要慢慢消化的，他识趣的没有多说什么。

...

“学长，你说我是不是不适合学音乐啊？昨天的元旦晚会又被我搞砸了...”

走出办公室，陈希柚郁闷的趴在了窗台上，语气中满是茫然。

苏屿川靠上栏杆，和她保持着礼貌距离，思索许久后开口道：“你会觉得学小提琴是很无聊的事吗？”

“从来不会。”，陈希柚轻轻摇了摇头，“我应该会觉得不学小提琴是很无聊的事。”

听完，苏屿川释然的笑了。

“那你毫无疑问的适合学音乐啊，只不过是差最后的一点勇气而已。”

“勇气？”，陈希柚喃喃道。

“嗯，一开始是不怕持续的疼痛的勇气，之后是不惧练习的枯燥的勇气，最后是不畏众人的目光的勇气，每个乐器都是这个学习过程。”

陈希柚歪了歪头，幽幽道：“学长，你知道女生最讨厌什么嘛？”

“啥？”，苏屿川有些摸不着头脑。

“莫名其妙说一堆大道理。”

“...”

陈希柚见苏屿川脸都尬黑了，终于是又笑了出来：“不过，谢谢你的风凉话，我记下了。”

....

班级内，刚跑完操的同学正热火朝天的扎成堆开茶话会，乐队七人组也不例外。

但今天，就连隔壁班沈音符和妄伽棠都来串了，一看就是在讨论大事。

“姓苏的，你可算是回来了！你知道嘛？视频的反响格外好呢！”

许昕念见苏屿川姗姗来迟也不恼，激动的拽着他的衣袖，把刚说过的话又复述了一遍。

大致意思就是，许昕念的视频剪的多么多么好，点赞多么多么多，评论多么多么棒。

“一条视频涨粉四位数呢！评论都在说什么——插个眼，以后火了一人一张亲签专辑。”，许昕念说着说着又笑了出来。

看得出来她是真的很开心。

“真有你的...”，苏屿川深吸了一口气。

“不过貌似有半成都是安安的颜值粉..嘿嘿。”

闻言，众人丝毫不感到意外，其他的什么都能反驳，但这点却反驳不了。

林穗安的颜值太抗打了，放哪都很惊艳，整个乐队也只有许昕念能与之比肩，但她作为幕后人员并不出镜。

虽然沈音符和江露白长得也都不赖，但更大的魅力还是在气质上，架不住林穗安是个超模怪。

更何况，主唱本身受到的关注度就是成员里最高的，也可以说是乐队的门面。

上述几点，苏屿川颇有体会。

毕竟林穗安的另一层马甲可是千万博主「秧秧」呢，硬是靠着颜值出圈的。

“既然这样，干脆小许也去露个脸吧？或许又能吸引一波颜值粉呢？！”，沈音符灵机一动道。

她一点嫉妒的意思都没有，毕竟林穗安的人格魅力就是那么大，被她压一头一点都不会让人感到不适。

“不了不了，说到底我就是个做幕后工作的，乐队主要营销的还是你们五位啊。”，许昕念连忙摆手推脱。

让她这个当经纪人的来消费成员热度，未免有喧宾夺主之嫌，毕竟他们的本意不是靠外观来赚人气，而是才华！

“视频号叫什么？我回家关注一下。”，听完了许昕念的慷慨陈词，余观棋也开始期待了起来。

“还能叫什么，StarryStar呗，简介也挂上了缩写「DOS」和中文名「繁星」，粉丝们对这些队名也都是一致好评呢。”，许昕念得意的不得了。

“有没有我的颜值粉啊？”

见林穗安如此受欢迎，苏屿川不禁有了危机感，他自问自己长得也不赖，便腆着张脸问道。

“有你大爷，你的黑粉倒是挺多的。”，许昕念白了一眼。

“不能吧？”，说这话的是林穗安，因为当事人苏屿川已经彻底呆住了。

许昕念噗嗤的笑了出来：“没办法呀队长，你的自我介绍装的没边了，弹奏的功底也不厚，观众自然要挑你的刺。”

“那很糟糕了，苏嘉豪。”，余观棋也忍不住嘲笑。

“虽然你可能不太想认识我，但我还是想介绍一下自己，结尾还来一个「嘿嘿嘿」，你不被喷谁被喷？”，江露白是看过完整视频的，这会儿也跟着打趣起来。

“屿川...你还好吧？”，妄伽棠见苏屿川一副生无可恋的模样不禁担心道。

“不好，很不好..虽然知道我弹的吉他就是一坨大粪，虽然知道我说话狂的没边...但没想到竟然真的被网暴了，事到如今只能把苏屿川这个人给销毁了，对了，有人教过我上吊...只要一根绳子就行，不痛的，哈哈..”

他延续了林穗安的说话方式——语速飞快，平静而又透露着浓烈的死感。

林穗安见状也是一脸的无奈，苏苏酱最近是越来越皮了。

“放心吧，你长得还算有人样，目前还没上升到人身攻击。”，许昕念补充说道，“压力就是动力呀，加油啊队长！”

“站着说话不屁股疼，说得倒是轻巧。”，苏屿川还是郁郁寡欢。

不过这会儿，众人的话题已经转移到了乐队整体上。

“就目前阶段，我想定制个队徽和队服，那样应该会更有乐队的感觉，只不过具体方案还没有想好。”，经纪人许昕念规划道。

闻言，江露白顿时兴奋了起来。

“我来我来！设计什么的我最在行了！”

她一直想为乐队做点事，这会儿总算是找到了机会。

“好，那就交给你了。”
