**第128章 来自班长大人的壁咚？！**

这些天，得益于万元存款的加持，苏屿川的腰杆都挺直了不少。

对此，感受最明显的就是妄伽棠。

“sugar，别做饭了，中午带你吃海鲜！”

“sugar，别买菜了，我点了外卖。”

“sugar，上号，我往你邮件里送了个皮肤。”

苏屿川的暴发户心理让妄伽棠倍感不适，三天两头的叮嘱他要注意节俭，生怕他掉进了消费陷阱。

毕竟有钱也禁不住这么花啊...

不过苏屿川一点都不上套，一心只顾着改善两人一猫的生活质量。

....

日子不知不觉就到了三月底。

冬装已经彻底换下，长袖加外套才是当季的标配。

教室外，柳垂桑茂，万顷春光无限明媚，教室内，伏案围桌，少年热血不休沸腾。

“苏苏酱，一模考怎么样？”

“650，比上次高6分。”

“棒棒哒，继续加油呀，你都跟学委并列第二啦！”

面对女神的鼓励，苏屿川并没有表现得多么欣喜，反倒是有些苦涩的摇了摇头。

20分的差距始终横亘在他与林穗安之间，近在咫尺却又遥不可及。

明明语文都能考到三位数了，但还是感到力不从心。

“话说，班长特地来找我，不能只是单纯问我的的一模成绩吧？”

“那个，苏苏酱，借一步说话。”

林穗安欲说些什么，望了望四周后，小心翼翼的牵起苏屿川的袖子，将他领到了室外。

这个隐蔽的连廊还真是好用，苏屿川已经不记得这是第几次和别人在这里说悄悄话了。

他默不作声，痴痴的望着面前的林穗安——这位女孩在半年前还是遥不可及的存在，现在却已经同自己这般亲密。

“半个月前答应我的事，还记得吧？”，林穗安用手臂将苏屿川逼至墙角，目光灼灼道。

苏屿川瞳孔地震——这..这这！这是壁咚啊！

这一定是壁咚吧？林穗安的两条胳膊可都按在墙上了啊，而且还把自己困住了啊！

虽然是进攻性十足的动作，但苏屿川却一点威胁都没有感受到。

毕竟班长比自己矮了小半个头，这会儿与自己对视都要仰头，玩壁咚这一套倒更像是在撒娇求抱抱。

小孩子似的。

“半个月前？什么事啊？”，苏屿川明知故问道。

闻言，林穗安踮起了脚尖，努力让自己的气势占据上风：“你答应了要陪我出cos的。”

苏屿川见她如此可爱，顿时萌生了欺负的念头。

轻轻往她踮起的脚尖上踢了踢，林穗安顺势一个踉跄跌倒在了地上。

她的壁咚无疾而终了。

“你干嘛！！”，林穗安摸了摸摔得生疼的大腿根，语气幽怨道。

苏屿川不以为意的耸了耸肩：“个子矮就别学人霸道女总裁玩壁咚，给你长个教训而已。”

“还有，你发我的cos服我看了，勉强能接受。”

“嘿嘿，那就行。”，林穗安听到这话一下子就开心了，一股脑爬了起来，“我就知道苏苏酱最好啦。”

苏屿川轻轻笑了笑：“毕竟那也算是我的入坑番，出个cos致敬一下也挺有意思的。”

“苏苏酱以前没出过cos嘛？”

林穗安默默的走到栏杆处，看了看楼下吐露新芽的树木，又理了理被风吹起的发丝。

苏屿川摇了摇头。

“很有意思的。”，林穗安转过头，向他示以甜甜的微笑，“你的头发长，盈盈姐还不用特意修假发，到时候我还可以给你化妆...”

“都行。”

“好呀，嘿嘿。”，林穗安兴致很高，心情也无比雀跃，“话说，不知不觉又到四月了呢。”

“嗯， 时间过得还真快啊。”，苏屿川叹了口气，“感觉啥都没学会就要高考了。”

“就是说啊...”

两人有一搭没一搭的聊着天聊着地，像是至交，亦像是挚友。

...

回到座位后，苏屿川这才开始大喘气——装正经也太他娘累了，刚才林穗安壁咚他的时候，他心脏都快飞出去了。

一想到刚才那暧昧的场景，苏屿川顿时又老脸一红。

哪有女孩子主动壁咚的啊？偏偏林穗安还是一副不自知的模样，这算哪门子事啊！她是玩galgame玩傻了吗？

唔...

他一个大男人竟然也像小女生那样害羞了起来。

龟男，这一波自己简直就是龟男！

幸好刚才及时打断施法，不然肯定要压不住枪。

...

这天晚自习结束，苏屿川照例和余观棋一起走，这样的日子从这学期开学一直持续到了现在。

苏屿川不知道余观棋是怎么想的，反正他有些怀念昔日三人无话不谈的场景。

“小样，你不能就这样和江露白闹到高考吧？至少朋友还得做啊。”

苏屿川原以为只是小打小闹，可二人这要死不活的模样已经保持了整整两个月了，皇上不急太监都要急。

“我道过歉了。”，余观棋语气平淡道。

闻言，苏屿川叹了一口气。

两位笨蛋发小都很强势，谁也不愿意做最先低头的那一个，可越是这样矛盾越是会加剧。

“小样，你知道江露白这次一模考的怎么样吗？”，沉默片刻后，苏屿川再次开口道，“反正在我印象里，她还没考过这么差，当然，我倒不会说这是你的原因，我还没有那么极端。”

“但是你要知道，她是喜欢你的啊，她肯定也不希望你和她闹成这样。”

余观棋怔了怔，旋即加快了脚步，似乎是不想再听苏屿川唠叨。

虽然苏屿川的话里并没有责怪自己的意思，但余观棋还是审视起了自己。

...

江的成绩虽然不如他们二人，但也是一直能高出特控线五六十的，这回倒好，差点连特控线都没够着。

换作往常，她的成绩但凡有一点点退步都要找自己抱怨几句，可这回却安静得很，至始至终都是一副波澜不惊的模样，像是接受了这个惨淡的结局。

这一点都不像她。

虽然江露白嘴上一直说着不在乎成绩，但其实她背地里也有在努力，也会向自己诉说压力。

最直观的，她每次考完试都要找自己对一下答案。

后方

苏屿川见余观棋又要回避，再也忍无可忍，快步追上去扯住了他的书包，满脸严肃道：“余观棋！你还要躲到什么时候？老子他妈的都快看不下去了，大男人低个头有那么难吗？”

余观棋愣住了——苏屿川很少喊自己的全名，除非他的态度认真到了一定地步。

换做往常，余观棋还愿意陪发小说几句掏心窝子的话，但现在的他烦的没边。

“我凭什么低头？苏屿川，你自己都当了三年的缩头乌龟，现在又有什么资格指责我？”，余观棋用尽全力推回了苏屿川的手臂，下一瞬，又狠狠的反拽住他的衣领。

“老子爱干啥干啥，江露白怎么样与我无关！你喜欢她你自己表白去！别扯我！”

“...”

苏屿川低下了头，但这并不是要避余观棋的锋芒，而是在暴风雨来临前最后的宁静。

从小到大，两人的关系一直很好，像这样激烈的争执从来没有过。

但没有过，不代表不会有。

“余观棋！老子操你大爷的！”，苏屿川眼睛气得猩红，缩头前冲将余观棋顶到了地上。

余观棋自知说了很过分的话——林穗安是苏屿川的逆鳞，自己先前那样羞辱他，他不可能不生气。

正因如此，被苏屿川推倒时，余观棋一点反抗都没有。

被好兄弟打一顿也挺好的，他脑子里一团乱麻。

不出所料，迎接他的，是苏屿川悬停在半空的拳头，不过这个拳头终究是没有落下去。

苏屿川再怎么不爽，也不会真的和余小样动真格。

“一句话，你到底喜不喜欢江露白。”

“...”，余观棋双眼无神的看着天空中的夜幕。

寂静又可怖。

这个问题他像是想了一个世纪之久，又像是脱口而出，总之，他感受不到时间的流逝，而苏屿川也就这样陪他耗着。

“喜欢。”

最终，余观棋给出了他的答案。
