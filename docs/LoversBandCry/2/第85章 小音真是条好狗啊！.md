**第85章 小音真是条好狗啊！**

这周六，乐队的例行训练取消了。

难道是人员持续调动，陷入了解散危机？

难道是狗血剧情上演，队友们勾心斗角？

...怎么想这些都不可能会发生的，不过是要准备期末考试罢了。

...

考试是在下周一到周三，这个双休的时间是完全自由的，也就是说复习与否全靠自觉。

可这毕竟是高三上学期的最后一次考试，要说毫不重视，想必鬼都不会信。

外面的气温又降了，光是出门都很艰难。

这会儿，就连一向精力过剩的苏屿川都自觉的宅在家复习。

看了看桌子上堆的老高的《优秀作文集选》，苏屿川不禁头大，离了林穗安以后，他真是一点学语文的劲头都没有。

期中好不容易才把语文考及格，可这些天沉迷吉他，又给荒废了，照这样下去，成绩出来后怕是要被李杰拎起来凌迟处死。

转头看看旁边的妄伽棠——他正戴着耳机，沉浸的温习着错题。

妄伽棠一直从早上八点钟坐到了现在，除了喝水和上厕所之外动都没动过，简直不像正常人。

除了学吉他，苏屿川做什么事都安静不下来，这会儿已经是憋得浑身都难受了。

“sugar，我出去透透气。”

他用力做了个深呼吸，丢下这句话便出门去了。

午饭的时间已经过了，纵使当前气温已经是全天最高，但手还是只能揣在口袋里。

“分开时难过不能说”

“谁没谁不能好好过”

“那天我们走了很久”

“没有争吵过”

苏屿川戴上了帽子和有线耳机，漫无目的的走在街道里，由于大降温，哪怕是在周末，路上也只有寥寥几人。

官亭巷，这里是庐州有名的小吃街 ，不知不觉间就逛到了这里，望着两侧林立的餐饮店，苏屿川的肚子竟丢人的叫了起来 。

为了专注复习，他和妄伽棠的午饭只吃了泡面，既然来都来了，苏屿川便想着买些带回去给妄伽棠加餐。

“两杯奶茶，热的，不要加小料。”

点好后，苏屿川走进店内蹭暖气，好巧不巧与陈佳佳打了个照面。

她貌似也是在等待出餐，与两手空空的街溜子苏屿川不同，陈佳佳此时背着书包，一看就知道是个书生。

可能是龚玥前些天说过陈佳佳对自己的态度不一般，也可能是自己生病那会儿受到了她的照顾，这会儿苏屿川见到陈佳佳难免有些尴尬。

“学委这是学习学累了来学做奶茶吗？”，苏屿川半开玩笑道。

陈佳佳见来人是苏屿川，瞳孔不着痕迹的放大了几分，沉默好一会儿才做出回应：“没有，我刚从图书馆出来，现在准备回家了。”

“图书馆？离这里不是很远吗？学委好高的兴致啊。”

“就因为图书馆远才去的啊，想着「这么远来都来了，不好好学习就浪费了」这种话来催眠自己。”，陈佳佳这会儿的语气也自然了许多，一如在学校般平易近人。

苏屿川闻言不禁笑了起来：“平时怎么不见你这么幼稚。”

“...”，陈佳佳有些不自然的扭过了头，由于她的个子高，苏屿川看不清她脸上的表情。

话说回来，貌似林穗安每次别扭转头的时候，少女的娇羞他总是能尽收眼底。

“苏同学，你和班长他们组了乐队是吗？”，稍微平复了一下心情后，陈佳佳向他递出了询问的目光。

苏屿川点了点头。

“视频我看过了，你吉他弹的真的很不错。”

其实陈佳佳也听到了评论区的一些质疑的声音，可安慰的话就在嘴边，却始终都说不出去。

她想去安慰，却不知该以什么立场。

“谢谢。”，苏屿川冲她笑了笑。

“其实...算了，没事，那个，我的奶茶好了，先走了，期末考加油。”，陈佳佳一副欲言又止的模样，最后悻悻的走了出去。

“她咋了？”，苏屿川看着她的背影，忍不住嘟囔道。

“...”

走出奶茶店后，陈佳佳低头走着，离苏屿川越远，她的脚步就越快，最后更是小跑了起来。

是店里的暖气开太高了嘛？怎么自己的脸到现在还烫烫的。

与他独处时，为什么自己的心跳总是那么快？大脑还会时不时的一片空白？

明明自己与他没有过多的交集，可每看到他就再也移不开视线，想去了解一切与他有关的人或事。

...

“下雪了啊？”，苏屿川离开官亭巷后，抬头看着天空中飘落的雪花，不禁有些呆滞。

上高中到现在，这还是第一次下雪。

虽然是初雪，但来得是真迟，现在都快一月中旬了。

雪花越下越大，丝毫没有减弱的趋势，原本沉寂的路上此刻也多了几分喧嚣。

靠近南方，人们对洁白的雪花总是心怀喜爱的，更何况还是间隔两年没下雪，一下便有如此的规模。

实在是稀罕事。

苏屿川不禁想起了他学的第一篇文言文。

白雪纷纷何所似？

如此灿烂盛大，不像粗盐，也不像柳絮，反倒像是天使抖落了她神圣的羽毛。

...

等他到家时，雪花已经在地上积了一层。

“下雪了吗？”，妄伽棠注意到了苏屿川身上的积雪。

“嗯，规模空前的大，今年冬天应该挺冷的。”，苏屿川递过一杯奶茶到他手边，语气漫不经心。

“这几天出门注意安全，容易摔倒。”

妄伽棠看起来对窗外的雪花毫不眷恋。

...

【穗穗安】：对呀，我刚才一直在看书，妈妈喊我吃晚饭才看到呢。

【穗穗安】：小音也很兴奋呢，一直在院子里乱转，吃完饭我想带它出门转转，苏苏酱要一起嘛(〃ﾟ3ﾟ〃)？

“sugar，你看我穿的这身有没有暖男的感觉？”

出门前，苏屿川特地打扮了一番，看起来人模狗样的。

“又要出门？”，妄伽棠狐疑道。

“踏雪寻梅嘛。”，苏屿川嘿嘿笑道。

“没关系，不错就行，我要风度不要温度。”
