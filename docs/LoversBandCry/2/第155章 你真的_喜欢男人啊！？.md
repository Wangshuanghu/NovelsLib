**第155章 你真的..喜欢男人啊！？**

“小沈..你这次不是考得挺好的吗？怎么又要我帮你做笔记啊？”

苏屿川不在家的这两天，妄伽棠也没有闲着——沈音符三天两头的喊他去图书馆学习，每次都是以“帮我巩固知识点”为由。

可小沈问的那些就算考到了她也都会啊..都快满分了...

如果这事让苏屿川知道了，他一定会狠狠的啐二人一口——他俩总分加在一块都快干到一千四了啊！

约会就约会，还非要以学习为幌子，找了个冠冕堂皇的借口。

简直是玷污了“知识点”和“图书馆”。

“还不够。”，沈音符轻轻的叹了口气，“离我想上的大学还差一点。”

妄伽棠有些不明所以——670多分已经很够用了，报考绝大多数的高校都不在话下，小沈的追求到底是有多高啊？

“能问一下你想上哪个大学吗？”

“不说。”

沈音符在妄伽棠面前总会展现出强势的一面。

在她看来，自己和妄伽棠的关系就像弹簧，只要她用力施压，妄伽棠就只能被牵着鼻子走。

大女子主义就是这样的！

妄伽棠见沈音符不想说，很识趣的没有再继续追问。

“班长，我觉得你有点反差萌。”

说是要潜心学习，但只要和妄伽棠坐在一起，沈音符根本顾不上其他，一个劲的想着要找话题。

好不容易把他骗出来，不好好调教怎么行？

调教他可比写题目有意思多了！

“反差？萌？”，妄伽棠脸上大写着困惑。

“对呀，反差萌。”，沈音符左手托着嘴巴，右手拿笔戳了戳妄伽棠的脸颊，以一副上位者的姿态说道：“昨天下午表演时鼓都要被你敲烂了，我在你旁边都怕被炸飞呢！”

妄伽棠挠了挠头，尴尬的开口道：“对不起，当时太激动了..有点没收住。”

“不许道歉！”，沈音符手上的力度加大了几分，笔帽在妄伽棠的脸颊压出了深深的窝，“你没做过任何错事，不许向任何人道歉。”

“好..好的。”，妄伽棠呆呆的望着她，见沈音符一脸认真，不禁在心里泛起了嘀咕——小沈还是个直爽的女孩子啊，坦率真诚，表里如一，心里也藏不住事。

空气安静了片刻。

“那个..班长..你觉得和我做同学烦不烦啊？”，沈音符收回了笔，有些扭捏的问道。

放在以前，她觉得“娇羞”不像是会出现在自己身上的词汇，但在妄伽棠的面前，她偏会显露出小女生的一面。

妄伽棠认真的摇了摇头回答道：“从来没有过，不如说我很荣幸能够认识你，很荣幸能和你做同学和朋友。”

小沈就像是自己的田螺姑娘，她会收拾自家荒芜的庭院，也会安抚自己孤寂的心灵。

“那就好。”，沈音符心满意足的笑了，低下头后喃喃道：“那就好...”

她清楚的记得妄伽棠当初说过要考江宁大学，为此，沈音符专门查了往年的投档线，知道高考要考到680分才有机会。

她想继续追随着妄伽棠的脚步。

好像从喜欢上他以来，追随他便成了自己高中日常的主旋律...无论是在学习上，生活上，还是在音乐上。

渐渐的，沈音符都不记得自己是怎么喜欢上妄伽棠的了。

伙伴们都知道她喜欢，却不知道她为何喜欢。

沈音符自问自己是个愚钝的人，在生活中也没什么主见，唯一擅长的就是迎合别人。

比文文静静的林穗安要开朗些，比大大方方的江露白要内敛些——她感觉自己就是大街上最不起眼的路人甲。

沿着父母划定的人生轨道，沿着主流造就的既定路线，小小的她一路成长到了现在。

除了成绩，自己似乎没有什么拿得出手的天赋。

...

“班长...”，沈音符做了个深呼吸，再次抬头同妄伽棠说起了话，“你知道我对你的第一印象是什么嘛？”

妄伽棠摇了摇头。

“「这家伙好烦，为什么又考了第一名？」”，虽然沈音符的语气稍显夸张，但却说得云淡风轻，“因为在你转学来之前，我一直都是班级第一嘛，所以第一名被你抢走之后，一开始我是很不服气的。”

“但我现在想开了，对你心服口服。”

沈音符坚信她不是会一见钟情的恋爱脑女生，那又是怎样喜欢上妄伽棠的呢？

难道自己是智性恋？

“哈哈...谢谢你的夸奖，不过我也是机缘巧合下才来的庐州，抢了你的第一名不是我的本意。”，妄伽棠笑了笑。

“不不不！”，沈音符怕他误会，急忙摆了摆手，“请你务必，一定，以及肯定！要一直当第一名！”

...好吧，自己就是慕强，对班长的喜欢就是由“不甘心”发展来的。

...

两人一直学习到了黄昏，妄伽棠收获颇丰，沈音符却没怎么把书看进去。

她的心有点乱。

她固然知道临近高考有这样乱七八糟的念头是很不合适的，但一想到毕业后或许就会分道扬镳，她怎么也按捺不住内心汹涌的感情。

“班长，你觉得我是个什么样的女生呢？”，前往地铁站的路上，沈音符故作无所谓的朝他问道。

“...很努力，很活泼，很会照顾人。”，妄伽棠思虑了片刻。

“那你会喜欢这样的女生吗？”，沈音符认真的看向了他。

“...”，妄伽棠以沉默回避了这个问题，“我应该不会喜欢女生的。”

闻言，沈音符直接吓得瘫在了地上。

“你..你真的..喜欢男人啊？”

不！还我文静温柔的班长！不要啊！

“我不是那个意思。”，妄伽棠轻轻扶着她站了起来，“只是我觉得我没资格喜欢人。”

“为什么？”，沈音符委屈道。

她可不擅长憋着自己的感情，这些天都反反复复的暗示好多回了，可班长的心还是跟冰淇淋似的。

一点也不开心。

“爸爸生前说过，男人要有担当，我现在还负不了责任，所以不能喜欢人。”，妄伽棠诚恳的回答道。

“责任和担当就那么重要吗？”，沈音符还是苦着张脸。

“重要。”，妄伽棠表情十分严肃，“「自我价值的实现依托责任存在，人生的宽度应用担当衡量」，这是爸爸经常挂在嘴边的话，所以在我看来，这二者比生命还重要。”

“可我不喜欢听大道理。”，沈音符用力的捂住了耳朵。

“那如果有个女生愿意一直陪着你呢？她会选择毫无保留的相信你，追随你，直到你能实现你所说的担当与责任，那你现在可以喜欢她吗？”

沈音符觉得这话已经不能算是暗示，自己差点就把“我喜欢你”写在脸上了。

“我..做不到。”，妄伽棠低下了头。

他莫名的执拗，始终过不了心里的那关。

“妄伽棠。”，沈音符彻底心灰意冷，“我向你走了九十九步，脚都要踩你鞋上了，你为什么连最后的一步都不肯走呢？”

她倔强的抬起了头，任由眼泪在眼眶里打转。

“如果这是古诗文，我已经背到了最后一句，如果这是数学题，那我只剩下「综上可得」还没写。”

“如果这是英语，应该是等着最后涂答题卡...如果这是物理，从头到尾要用的公式我也都写上了。”

“可你呢？...你却告诉我我连准考证都没带？”

不争气的泪水还是流了下来。

“对不起..”，哪怕小沈刚才还让自己别说「对不起」，但此时妄伽棠除了道歉不知道还能做些什么。

“妄伽棠，你从来就没把我当回事吧？”，沈音符认命，开始破罐子破摔，“别告诉我你不知道我喜欢你，回应我就这么难吗？你真的就对我一点感情都没有吗？”

“...”，哪怕她拽着自己的衣领，妄伽棠也没有开口应答。

他坚定的认为「喜欢」和「爱」一辈子只能对一个人说，如果自己也能像屿川那般阳光开朗，家庭幸福，那一定能给小沈想要的答案。

但没有如果，自己只是个没有依靠没有背景的孤儿，甚至就连未来在哪都看不清。

“我真是个笨蛋，我为什么会喜欢上你啊？我到底是喜欢你什么啊？”，沈音符越哭越凶，大颗大颗的泪珠跟不要钱似的倾落，“你为什么要那么优秀？为什么要让我喜欢上你？”

“为什么要邀请我加入乐队？为什么要教我做饭？为什么对我温柔，让我觉得我于你而言是特殊的存在？”

“为什么？为什么啊？”

妄伽棠如提线木偶般任由她摆弄，喉咙和四肢就像是被灌了铅，什么都说不出，什么都做不了。

回过神来时，小沈已经跑远了。

自己还是第一次惹女孩子哭，可为什么哭的是她，自己却那么难过呢？

...

【忘加糖】：我惹哭了一位很重要的人，我很难过，应该怎么做？

地铁上，沈音符没想到这会儿妄伽棠还有闲心来找自己取经，差点就没忍住自爆马甲。

惹哭了自己，还问自己应该怎么做？

【不太甜】：她是你的什么人呢？你又把她当什么呢？

妄伽棠并没有因这近乎苛责的话起疑心，认认真真的敲下了八个字。
